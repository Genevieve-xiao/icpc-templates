# icpc-templates

这是一个用于整理算法竞赛模板、专题讲解和刷题经验的个人仓库。

目前上传的是之前已经写过的一部分内容，后续会随着刷题、复习和博客更新继续补充。

## 目录结构

```text
icpc-templates/
├── 专题/
│   ├── 并查集
│   ├── 差分
│   ├── 浅谈异或
│   └── 网络流
├── 算竞板子
└── 算竞小技巧
````
