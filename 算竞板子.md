
# 基础算法
## 二分


```cpp
lower_bound -> 第一个 >= x
upper_bound -> 第一个 >  x
upper_bound - lower_bound -> x 的出现次数
pos!=end && *pos==x -> 判断 x 是否存在
```

最后一个满足 check 的数，区间要向右收拢，所以是 `l = mid`

```cpp
 while(l < r){
     int mid = (l + r + 1)>>1;
     if (check(mid)) l = mid;
     else r = mid - 1;
 }
```

第一个满足 check 的数，区间要向左收拢，所以是 `r = mid`

```cpp
 while(l<r){
     int mid = (l + r) >> 1;
     if (check(mid)) r = mid;
     else l = mid + 1;
 }
```

如果是 `l = mid`， 那么接下来就是 `r = mid - 1`，而且前面的 `mid` 要补加1；
如果是 `r = mid`，那么接下来就是 `l = mid + 1`。

最终停止条件是 `l == r`。
## 排序
### 手写快排

快排的基本思想是**分治**。时间复杂度为 $O(nlogn)$，是一种十分理想的排序方式，通过c++的 stl 可以直接使用 sort 来实现。手写快排时常常会遇到**边界问题**，需要特别注意。
##### 步骤：
- 每次在当前区间中选择一个基准数x，x可以为```q[l],q[r],q[l+r>>1]```
- 选取两个指针分别从区间的左右端点开始，左指针i遇到第一个大于等于基准数的值就停下，右指针j遇到第一个小于等于基准数的值就停下，然后交换这两个值
- ij指针按照上部规则继续向前走，直到$i>=j$停止循环
- 这样我们就实现了将一个待排序区间分成了左右两个区间，左区间全部满足小于等于基准数，右区间全部满足大于等于基准数，分治思想由此体现
- 接下来递归执行 $[l,j]$ $[j+1,r]$ 这两个区间，（当然也可以以i为分界点，但是可能会出现边界问题，用j更好一些）

```cpp
void quick_sort(int q[],int l,int r){
	if(l>=r) return;
	int x=q[l+r>>1],i=l-1,j=r+1;
	while(i<j){
		do i++; while(q[i]<x);
		do j--; while(q[j]>x);
		if(i<j) swap(q[i],q[j]);
	}
	quick_sort(q,l,j);
	quick_sort(q,j+1,r);	
}
```

### 归并排序
归并排序的基本思想也是分治。时间复杂度是严格的 $nlogn$
##### 步骤：
- 确定分界点 `mid=l+r>>1`
- 将分成的两个区间分别递归
- 双指针分别指向区间最小值，然后依次将该值放入暂存数组tmp[]中
- 将tmp中的数据覆盖到原数组中

```cpp
void merge_sort(int q[],int l,int r){
    if(r<=l) return ;
    int mid=r+l>>1;
    merge_sort(q,l,mid);
    merge_sort(q,mid+1,r);
    
    int k=0,i=l,j=mid+1;
    while(i<=mid&&j<=r) 
        if(q[i]<q[j]) tmp[++k]=q[i++];
        else tmp[++k]=q[j++];
    while(i<=mid) tmp[++k]=q[i++];
    while(j<=r) tmp[++k]=q[j++];
    
    for(i=l,j=1;i<=r;i++,j++) q[i]=tmp[j];
}
```

稳定排序是指原来相等的两个元素前后相对位置在排序后依然不变。
快排是不稳定的，归并是稳定的。

### 扩展欧几里得 Exgcd

**作用**：求整数 `x, y`，使得 `a * x + b * y = gcd(a, b)`。

**输入**：两个整数 `a, b`。

**输出**：`gcd(a, b)`，以及一组系数 `x, y`。

**相关知识**：  
扩展欧几里得常用于求模意义下的乘法逆元。若 `gcd(a, mod) = 1`，则存在 `x` 使得 `a * x ≡ 1 (mod mod)`，此时 `x` 就是 `a` 在模 `mod` 意义下的逆元。答案通常要写成 `(x % mod + mod) % mod`，避免负数。

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
ll a, b, x, y;
// 求 x, y，使得 a * x + b * y = gcd(a, b)
ll exgcd(ll a, ll b) {
    if (!b) {
        x = 1;
        y = 0;
        return a;
    }
    ll d = exgcd(b, a % b);
    ll tx = x;
    x = y;
    y = tx - a / b * y;
    return d;
}
int main() {
    cin >> a >> b;
    ll d = exgcd(a, b);
    cout << d << '\n';
    cout << x << ' ' << y << '\n';
    return 0;
}
```

### 快速幂

**作用**：快速计算 `b^p mod k`。

**输入**：三个整数 `b, p, k`，表示底数、指数、模数。

**输出**：`b^p mod k`。

**相关知识**：  
快速幂利用指数的二进制拆分，把普通的 `O(p)` 次乘法优化到 `O(log p)`。  
例如 `13 = 1101₂`，所以 `b^13 = b^8 * b^4 * b^1`。

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
ll b, p, k;
// 计算 b^p mod k
ll fpow(ll b, ll p, ll k) {
    ll ans = 1 % k;
    ll base = b % k;
    while (p > 0) {
        if (p & 1) ans = ans * base % k;
        base = base * base % k;
        p >>= 1;
    }
    return ans;
}
int main() {
    cin >> b >> p >> k;
    ll s = fpow(b, p, k);
    printf("%lld^%lld mod %lld=%lld", b, p, k, s);
    return 0;
}
```

### 乘法逆元 
#### 乘法逆元：线性递推

**作用**：在线性时间内求出 `1 ~ n` 在模 `p` 意义下的乘法逆元。

**输入**：两个整数 `n, p`。

**输出**：依次输出 `1, 2, ..., n` 在模 `p` 意义下的逆元。

**相关知识**：  
该方法常用于一次性求很多个逆元，复杂度为 `O(n)`。  
常见适用条件是 `p` 为质数，且 `1 <= n < p`。  
递推公式为：

```text
inv[1] = 1
inv[i] = p - (p / i) * inv[p % i] % p
```
```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
const int N = 5000010;
ll n, p;
ll inv[N];
// 线性递推求 1 ~ n 在 mod p 下的逆元
int main() {
    cin >> n >> p;
    inv[1] = 1;
    printf("1\n");
    for (int i = 2; i <= n; i++) {
        inv[i] = p - (p / i) * inv[p % i] % p;
        printf("%lld\n", inv[i]);
    }
    return 0;
}
```

#### 乘法逆元：费马小定理

**作用**：用快速幂求 `1 ~ n` 在质数模数 `p` 下的乘法逆元。

**输入**：两个整数 `n, p`。

**输出**：依次输出 `1, 2, ..., n` 在模 `p` 意义下的逆元。

**相关知识**：  
费马小定理：
```text
若 p 是质数，且 gcd(a, p) = 1，则 a^(p - 1) ≡ 1 (mod p)
```
所以：
```text
a 的逆元为 a^(p - 2) mod p
```
该方法复杂度为 `O(n log p)`，适合求单个逆元或少量逆元。若要求 `1 ~ n` 的全部逆元，通常线性递推更快。
```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
ll n, p;
// 计算 b^e mod p
ll fpow(ll b, ll e) {
    ll ans = 1 % p;
    ll base = b % p;
    while (e > 0) {
        if (e & 1) ans = ans * base % p;
        base = base * base % p;
        e >>= 1;
    }
    return ans;
}
int main() {
    cin >> n >> p;
    for (int i = 1; i <= n; i++) {
        printf("%lld\n", fpow(i, p - 2));
    }
    return 0;
}
```

#### 乘法逆元：Exgcd 做法

**作用**：用扩展欧几里得求 `1 ~ n` 在模 `p` 意义下的乘法逆元。

**输入**：两个整数 `n, p`。

**输出**：依次输出 `1, 2, ..., n` 在模 `p` 意义下的逆元；若某个数不存在逆元，输出 `No inverse`。

**相关知识**：  
该方法不要求 `p` 是质数，只要求当前数字 `i` 与 `p` 互质，即 `gcd(i, p) = 1`。  
若 `i * x + p * y = 1`，则 `i * x ≡ 1 (mod p)`，因此 `x` 是 `i` 的逆元。

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
ll n, p;
ll x, y;
// 求 x, y，使得 a * x + b * y = gcd(a, b)
ll exgcd(ll a, ll b) {
    if (!b) {
        x = 1;
        y = 0;
        return a;
    }
    ll d = exgcd(b, a % b);
    ll tx = x;
    x = y;
    y = tx - a / b * y;
    return d;
}
int main() {
    cin >> n >> p;
    for (int i = 1; i <= n; i++) {
        ll d = exgcd(i, p);
        if (d != 1) {
            cout << "No inverse\n";
        } else {
            x = (x % p + p) % p;
            cout << x << '\n';
        }
    }
    return 0;
}
```

### 有理数取余

**作用**：计算有理数 `a / b` 在模 `mod` 意义下的值，即 `a * inv(b) mod mod`。

**输入**：两个整数 `a, b`，整数可能很大，不能直接用 `long long` 保存。

**输出**：若 `b` 在模 `mod` 意义下存在逆元，输出 `a / b mod mod`；否则输出 `Angry!`。

**相关知识**：  
模意义下不能直接做除法，需要转化为乘法逆元：
```text
a / b ≡ a * b^(-1) (mod mod)
```
当且仅当 `gcd(b, mod) = 1` 时，`b` 的逆元存在。这里 `mod = 19260817` 是质数，因此只要 `b % mod != 0`，逆元就存在。
输入数字很大时，只需要边读边取模：

```text
x = (x * 10 + digit) % mod
```
```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
const ll mod = 19260817;
ll a, b;
// 读取大整数，并返回其对 mod 取模后的结果
ll read_mod() {
    ll x = 0, f = 1;
    int ch = getchar();
    while (ch != '-' && (ch < '0' || ch > '9')) ch = getchar();
    if (ch == '-') {
        f = -1;
        ch = getchar();
    }
    while (ch >= '0' && ch <= '9') {
        x = (x * 10 + ch - '0') % mod;
        ch = getchar();
    }
    x = x * f % mod;
    return (x + mod) % mod;
}
// 求 x, y，使得 a * x + b * y = gcd(a, b)
ll exgcd(ll a, ll b, ll &x, ll &y) {
    if (!b) {
        x = 1;
        y = 0;
        return a;
    }
    ll d = exgcd(b, a % b, x, y);
    ll tx = x;
    x = y;
    y = tx - a / b * y;
    return d;
}
int main() {
    a = read_mod();
    b = read_mod();
    ll x, y;
    ll d = exgcd(b, mod, x, y);
    if (d != 1) {
        printf("Angry!");
        return 0;
    }
    x = (x % mod + mod) % mod;
    printf("%lld", a * x % mod);
    return 0;
}
```

### 中国剩余定理 CRT

**作用**：求同余方程组的最小非负整数解。

```text
x ≡ b[1] (mod a[1])
x ≡ b[2] (mod a[2])
...
x ≡ b[n] (mod a[n])
```
**输入**：第一行输入 `n`，接下来 `n` 行每行输入 `a[i], b[i]`，表示 `x ≡ b[i] (mod a[i])`。

**输出**：输出方程组在模 `M = a[1] * a[2] * ... * a[n]` 意义下的最小非负解。

**相关知识**：  
普通 CRT 要求所有模数 `a[i]` 两两互质。设：
```text
M = a[1] * a[2] * ... * a[n]
m[i] = M / a[i]
```
需要找到 `m[i]` 在模 `a[i]` 意义下的逆元 `inv[i]`，则答案为：
```text
ans = Σ b[i] * m[i] * inv[i] (mod M)
```
如果模数不两两互质，应使用扩展 CRT，而不是普通 CRT。

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
const int N = 100010;
ll n, M = 1;
ll a[N], b[N];
// 求 x, y，使得 a * x + b * y = gcd(a, b)
ll exgcd(ll a, ll b, ll &x, ll &y) {
    if (!b) {
        x = 1;
        y = 0;
        return a;
    }
    ll d = exgcd(b, a % b, x, y);
    ll tx = x;
    x = y;
    y = tx - a / b * y;
    return d;
}
// 防止乘法中间结果溢出
ll mul_mod(ll a, ll b, ll mod) {
    return (__int128)a * b % mod;
}
// 普通 CRT，要求所有模数两两互质
ll CRT() {
    ll ans = 0;
    for (int i = 1; i <= n; i++) {
        ll m = M / a[i];
        ll x, y;
        ll d = exgcd(m, a[i], x, y);
        if (d != 1) return -1;
        x = (x % a[i] + a[i]) % a[i];
        ll t = mul_mod(m, x, M);
        t = mul_mod(t, b[i], M);
        ans = (ans + t) % M;
    }
    return (ans + M) % M;
}
int main() {
    scanf("%lld", &n);
    for (int i = 1; i <= n; i++) {
        scanf("%lld%lld", &a[i], &b[i]);
        b[i] = (b[i] % a[i] + a[i]) % a[i];
        M *= a[i];
    }
    ll ans = CRT();
    if (ans == -1) printf("No solution");
    else printf("%lld", ans);
    return 0;
}
```

### 线性筛

**作用**：在线性时间内筛出 `1 ~ n` 中的所有质数。

**输入**：一个整数 `n`。

**输出**：示例代码中输出 `1 ~ n` 的质数个数和所有质数。实际做题时通常直接使用 `prime[]` 数组和 `cnt`。

**相关知识**：  
线性筛的核心是：每个合数只被它的最小质因子筛掉一次，所以复杂度是 `O(n)`。  
关键语句：

```cpp
if (i % prime[j] == 0) break;
```
含义是：当 `prime[j]` 已经是 `i` 的最小质因子时，`i * prime[j]` 已经被最小质因子筛掉，后面不再继续枚举更大的质数，否则会重复筛。

```cpp
#include <bits/stdc++.h>
using namespace std;
const int N = 10000010;
int n, cnt;
int prime[N];
bool is_composite[N];
// 线性筛：筛出 1 ~ n 的所有质数
void sieve(int n) {
    is_composite[1] = true;
    for (int i = 2; i <= n; i++) {
        if (!is_composite[i]) prime[++cnt] = i;
        for (int j = 1; j <= cnt && 1LL * i * prime[j] <= n; j++) {
            is_composite[i * prime[j]] = true;
            if (i % prime[j] == 0) break;
        }
    }
}
int main() {
    scanf("%d", &n);
    sieve(n);
    printf("%d\n", cnt);
    for (int i = 1; i <= cnt; i++) {
        printf("%d%c", prime[i], i == cnt ? '\n' : ' ');
    }
    return 0;
}
```

### 求组合数
#### 组合数：阶乘逆元做法

**作用**：计算组合数 `C(n, m) mod mod`。

**输入**：两个整数 `n, m`。

**输出**：输出 `C(n, m) mod 1000000007`。

**相关知识**：  
组合数公式：
```text
C(n, m) = n! / (m! * (n - m)!)
```
在模意义下，除法要转化为乘逆元：
```text
C(n, m) ≡ fac[n] * invfac[m] * invfac[n - m] (mod mod)
```
因为 `1000000007` 是质数，所以能用费马小定理求逆元：
```text
a^(-1) ≡ a^(mod - 2) (mod mod)
```
该板子适合 `n` 较大、`mod` 为质数、且 `n < mod` 的情况。预处理复杂度为 `O(n)`，单次查询复杂度为 `O(1)`。

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
const int mod = 1000000007;
int n, m;
vector<ll> fac, invfac;
// 快速幂：计算 a^b mod mod
ll fpow(ll a, ll b) {
    ll ans = 1;
    a %= mod;
    while (b) {
        if (b & 1) ans = ans * a % mod;
        a = a * a % mod;
        b >>= 1;
    }
    return ans;
}
// 预处理阶乘和阶乘逆元
void init(int n) {
    fac.assign(n + 1, 1);
    invfac.assign(n + 1, 1);
    for (int i = 1; i <= n; i++)
        fac[i] = fac[i - 1] * i % mod;
    invfac[n] = fpow(fac[n], mod - 2);
    for (int i = n; i >= 1; i--)
        invfac[i - 1] = invfac[i] * i % mod;
}
// 计算 C(n, m) mod mod
ll C(int n, int m) {
    if (m < 0 || m > n) return 0;
    return fac[n] * invfac[m] % mod * invfac[n - m] % mod;
}
int main() {
    scanf("%d%d", &n, &m);
    init(n);
    printf("%lld", C(n, m));
    return 0;
}
```

#### 组合数：一维 DP 做法

**作用**：用杨辉三角递推计算 `C(n, m) mod mod`。

**输入**：两个整数 `n, m`。

**输出**：输出 `C(n, m) mod 1000000007`。

**相关知识**：  
递推式：

```text
C(i, j) = C(i - 1, j) + C(i - 1, j - 1)
```
一维滚动数组中需要倒序枚举 `j`，否则 `f[j - 1]` 会被本轮更新后的值污染。

该方法不依赖模数为质数，但复杂度是 `O(nm)`，只适合 `n * m` 较小的情况。

```cpp
#include <bits/stdc++.h>
using namespace std;
const int mod = 1000000007;
int n, m;
// 一维 DP 计算 C(n, m)
int C(int n, int m) {
    if (m < 0 || m > n) return 0;
    vector<int> f(m + 1, 0);
    f[0] = 1;
    for (int i = 1; i <= n; i++) {
        for (int j = min(i, m); j >= 1; j--) {
            f[j] += f[j - 1];
            if (f[j] >= mod) f[j] -= mod;
        }
    }
    return f[m];
}
int main() {
    scanf("%d%d", &n, &m);
    printf("%d", C(n, m));
    return 0;
}
```

### 康托展开

#### 朴素康拓展开

**作用**：求一个排列在所有排列中字典序第几小。

**输入**：一个长度为 `n` 的排列 `a[1...n]`。

**输出**：该排列的排名，下面板子输出的是从 `1` 开始的排名，即最小排列的排名为 `1`。

**相关知识**：  
康托展开的核心思想是：枚举每一位 `i`，统计这一位后面有多少个数比 `a[i]` 小，记为 `tmp`，那么以当前前缀为界，可以跳过 `tmp * (n - i)!` 个排列。

公式为：

```text
rank = 1 + Σ tmp[i] * (n - i)!
```
其中 `tmp[i]` 表示在 `a[i + 1...n]` 中小于 `a[i]` 的数的个数。

该朴素做法复杂度为 `O(n^2)`，适合 `n` 较小的情况。

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
const int N = 1000010;
const int mod = 998244353;
int n;
int a[N];
ll f[N]; // f[i] 表示 i! % mod
void ini() {
    f[0] = 1;
    for (int i = 1; i <= n; i++)
        f[i] = f[i - 1] * i % mod;
}
// 康托展开，返回排列 num[1...n] 的 1-indexed 排名
ll cantor(int num[]) {
    ll ans = 1;
    for (int i = 1; i < n; i++) {
        int tmp = 0; // tmp 表示后面比 num[i] 小的数的个数
        for (int j = i + 1; j <= n; j++)
            if (num[j] < num[i])
                tmp++;
        ans = (ans + tmp * f[n - i] % mod) % mod;
    }
    return ans;
}
int main() {
    scanf("%d", &n);
    ini();
    for (int i = 1; i <= n; i++)
        scanf("%d", &a[i]);
    printf("%lld", cantor(a));
    return 0;
}
```

#### 康托展开：树状数组优化

**作用**：用树状数组优化康托展开，求排列在所有排列中字典序第几小。

**输入**：一个整数 `n`，随后输入长度为 `n` 的排列。

**输出**：该排列的排名，下面板子输出的是从 `1` 开始的排名，即最小排列的排名为 `1`。

**相关知识**：  
朴素康托展开中，每一位都要向后扫描统计“小于当前数且还没用过的数”的个数，复杂度为 `O(n^2)`。

树状数组优化后，`v[i]` 不再存具体数字，而是存状态：
```text
v[i] = 1 表示数字 i 还没有被使用
v[i] = 0 表示数字 i 已经被使用
```
因此：
```text
getsum(x - 1)
```
表示当前还没有被使用、且小于 `x` 的数字个数。

每处理一个数字 `x` 后，需要执行：
```cpp
update(x, -1);
```
表示数字 `x` 已经被使用。

整体复杂度为 `O(n log n)`。

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
const int N = 1001000;
const int mod = 998244353;
ll n;
ll v[N]; // v[i] 表示数字 i 是否还未被使用，1 为未使用，0 为已使用
ll f[N]; // f[i] 表示 i! % mod
ll lowbit(ll x) {
    return x & (-x);
}
// 在树状数组中给位置 x 加上 k
void update(ll x, ll k) {
    for (ll i = x; i <= n; i += lowbit(i))
        v[i] += k;
}
// 查询 1 ~ x 中还未被使用的数字个数
ll getsum(ll x) {
    ll ans = 0;
    for (ll i = x; i; i -= lowbit(i))
        ans += v[i];
    return ans % mod;
}
void ini() {
    f[0] = 1;
    for (ll i = 1; i <= n; i++)
        f[i] = f[i - 1] * i % mod;
    for (ll i = 1; i <= n; i++)
        update(i, 1);
}
// 康托展开，边读入边计算 1-indexed 排名
ll cantor() {
    ll ans = 1;
    for (ll i = 1; i <= n; i++) {
        ll x;
        scanf("%lld", &x);
        ans = (ans + getsum(x - 1) * f[n - i] % mod) % mod;
        update(x, -1);
    }
    return ans;
}
int main() {
    scanf("%lld", &n);
    ini();
    printf("%lld", cantor());
    return 0;
}
```

#### 卡特兰数 Catalan Number

卡特兰数是一个数列，其前几项为（从第零项开始） : 
$1, 1, 2, 5, 14, 42, 132, 429, 1430, 4862, 16796, 58786, 208012, 742900, 2674440, 9694845, 35357670, 129644790, 477638700, 1767263190, 6564120420, 24466267020, 91482563640, 343059613650, 1289904147324, 4861946401452, …$

满足递推式
$$
\begin{aligned}
&C_{n+1}  
= C_{0}C_{n}+C_{1}C_{n-1}+...+C_{n}C_{0}\\
&(n-3)C_n 
= \frac{n}{2} ( C_{3}C_{n-1}+C_{4}C_{n-2}+...+C_{n-1}C_{3})\\
\end{aligned}
$$
$$
\begin{aligned}
f(n)
	&= \sum_{i=0}^{n-1} f(i)*f(n-i-1) \\
	& = \frac{C_{2n}^{m}}{n+1} \\
	& = C_{2n}^{n}-C_{2n}^{n-1}
\end{aligned}
$$

通过状态转移方程简单理解一下卡特兰数：
$$f[i][j]=f[i][j-1]+f[i-1][j]$$
输出递推矩阵。

#### 斯特林数 Stirling Number

**作用**：斯特林数常用于处理“把 `n` 个不同元素分成若干组”的计数问题。主要分为两类：第一类斯特林数表示把 `n` 个不同元素构成 `m` 个圆排列的方案数，第二类斯特林数表示把 `n` 个不同元素拆分成 `m` 个非空集合的方案数。原博客中也按这两类分别给出了含义和递推式。

**常见记号**：

```text
第一类斯特林数：
s1(n, m) 或 [n m]
表示 n 个不同元素构成 m 个圆排列的方案数。

第二类斯特林数：
s2(n, m) 或 {n m}
表示 n 个不同元素划分成 m 个非空集合的方案数。
```

**边界条件**：

```text
s(0, 0) = 1

当 n > 0 时：
s(n, 0) = 0

当 n >= 0 时：
s(n, n) = 1
```
##### 第一类斯特林数

**作用**：求 `n` 个不同元素构成 `m` 个圆排列的方案数。

**输入**：两个整数 `n, m`。

**输出**：输出第一类斯特林数 `s1(n, m) mod mod`。

**相关知识**：  
第一类斯特林数的递推式为：
```text
s1(n, m) = s1(n - 1, m - 1) + (n - 1) * s1(n - 1, m)
```
推导过程：
```text
考虑第 n 个元素放在哪里。

第一种情况：
第 n 个元素单独形成一个新的圆排列。
此时剩下 n - 1 个元素需要构成 m - 1 个圆排列。
方案数为 s1(n - 1, m - 1)。

第二种情况：
第 n 个元素插入到已有圆排列中。
此时前 n - 1 个元素已经构成 m 个圆排列。
在这些圆排列中，一共有 n - 1 个插入位置。
方案数为 (n - 1) * s1(n - 1, m)。

两种情况相加：
s1(n, m) = s1(n - 1, m - 1) + (n - 1) * s1(n - 1, m)
```
原博客中第一类斯特林数的含义和递推式也是按“`n` 个不同元素构成 `m` 个圆排列”来定义的。

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
const int N = 5010;
const int mod = 1000000007;
int n, m;
ll dp[N][N]; // dp[i][j] 表示 i 个不同元素构成 j 个圆排列的方案数
int main() {
    scanf("%d%d", &n, &m);
    dp[0][0] = 1;
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= min(i, m); j++) {
            dp[i][j] = (dp[i - 1][j - 1] + (ll)(i - 1) * dp[i - 1][j] % mod) % mod;
        }
    }
    printf("%lld", dp[n][m]);
    return 0;
}
```


##### 第一类斯特林数优化：一维滚动

**作用**：用一维数组求第一类斯特林数 `s1(n, m)`。

**输入**：两个整数 `n, m`。

**输出**：输出 `s1(n, m) mod mod`。

**相关知识**：  
因为 `dp[i][j]` 只由上一行的 `dp[i - 1][j - 1]` 和 `dp[i - 1][j]` 转移而来，所以可以滚动优化。

递推式：

```text
dp[j] = dp[j - 1] + (i - 1) * dp[j]
```
其中更新第 `i` 行时，`j` 必须倒序枚举，避免 `dp[j - 1]` 被本轮提前更新。

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
const int N = 1000010;
const int mod = 1000000007;
int n, m;
ll dp[N]; // dp[j] 表示当前 i 个元素构成 j 个圆排列的方案数
int main() {
    scanf("%d%d", &n, &m);
    dp[0] = 1;
    for (int i = 1; i <= n; i++) {
        for (int j = min(i, m); j >= 1; j--) {
            dp[j] = (dp[j - 1] + (ll)(i - 1) * dp[j] % mod) % mod;
        }
        dp[0] = 0;
    }
    printf("%lld", dp[m]);
    return 0;
}
```


##### 第二类斯特林数

**作用**：求 `n` 个不同元素拆分成 `m` 个非空集合的方案数。

**输入**：两个整数 `n, m`。

**输出**：输出第二类斯特林数 `s2(n, m) mod mod`。

**相关知识**：  
第二类斯特林数的递推式为：

```text
s2(n, m) = s2(n - 1, m - 1) + m * s2(n - 1, m)
```
推导过程：

```text
考虑第 n 个元素放在哪里。

第一种情况：
第 n 个元素单独形成一个新集合。
此时剩下 n - 1 个元素需要拆分成 m - 1 个非空集合。
方案数为 s2(n - 1, m - 1)。

第二种情况：
第 n 个元素放入已有集合中。
此时前 n - 1 个元素已经拆分成 m 个非空集合。
第 n 个元素可以放入其中任意一个集合。
方案数为 m * s2(n - 1, m)。

两种情况相加：
s2(n, m) = s2(n - 1, m - 1) + m * s2(n - 1, m)
```

原博客中第二类斯特林数被解释为集合拆分，并给出递推式 `s2(n,m)=s2(n-1,m-1)+m*s2(n-1,m)`。
```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
const int N = 5010;
const int mod = 1000000007;
int n, m;
ll dp[N][N]; // dp[i][j] 表示 i 个不同元素拆分成 j 个非空集合的方案数
int main() {
    scanf("%d%d", &n, &m);
    dp[0][0] = 1;
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= min(i, m); j++) {
            dp[i][j] = (dp[i - 1][j - 1] + (ll)j * dp[i - 1][j] % mod) % mod;
        }
    }
    printf("%lld", dp[n][m]);
    return 0;
}
```


##### 第二类斯特林数优化：一维滚动

**作用**：用一维数组求第二类斯特林数 `s2(n, m)`。

**输入**：两个整数 `n, m`。

**输出**：输出 `s2(n, m) mod mod`。

**相关知识**：  
二维递推：
```text
dp[i][j] = dp[i - 1][j - 1] + j * dp[i - 1][j]
```
滚动优化后：
```text
dp[j] = dp[j - 1] + j * dp[j]
```
同样需要倒序枚举 `j`，否则 `dp[j - 1]` 会被当前行提前更新。

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
const int N = 1000010;
const int mod = 1000000007;
int n, m;
ll dp[N]; // dp[j] 表示当前 i 个元素拆分成 j 个非空集合的方案数
int main() {
    scanf("%d%d", &n, &m);
    dp[0] = 1;
    for (int i = 1; i <= n; i++) {
        for (int j = min(i, m); j >= 1; j--) {
            dp[j] = (dp[j - 1] + (ll)j * dp[j] % mod) % mod;
        }
        dp[0] = 0;
    }
    printf("%lld", dp[m]);
    return 0;
}
```


##### 两类斯特林数对比

```text
第一类斯特林数：
把 n 个不同元素分成 m 个圆排列。
递推式：

第二类斯特林数：
```

#### KMP

**作用**：在线性时间内查找模式串 `p` 在文本串 `s` 中的所有出现位置，并求出模式串的 `ne` 数组。

**输入**：两个字符串 `s, p`，其中 `s` 为文本串，`p` 为模式串。

**输出**：先按行输出 `p` 在 `s` 中每一次出现的起始位置，再输出模式串 `p` 的 `ne[1...n]` 数组。

**相关知识**：  
`ne[i]` 表示 `p[1...i]` 中，最长的相等真前缀和真后缀的长度。

```text
ne[i] = max k，使得 p[1...k] = p[i-k+1...i]，且 k < i
```

KMP 的核心是：匹配失败时，模式串不从头开始，而是跳到 `ne[j]` 继续匹配。  
构造 `ne` 数组复杂度为 `O(n)`，匹配复杂度为 `O(m)`，总复杂度为 `O(n + m)`。

```cpp
#include <bits/stdc++.h>
using namespace std;
const int N = 1000100;
int ne[N];
char s[N], p[N]; // s 为文本串，p 为模式串
int n, m;
int main() {
    cin >> (s + 1) >> (p + 1);
    n = strlen(p + 1);
    m = strlen(s + 1);
    for (int i = 2, j = 0; i <= n; i++) {
        while (j && p[i] != p[j + 1]) j = ne[j];
        if (p[i] == p[j + 1]) j++;
        ne[i] = j;
    }
    for (int i = 1, j = 0; i <= m; i++) {
        while (j && s[i] != p[j + 1]) j = ne[j];
        if (s[i] == p[j + 1]) j++;
        if (j == n) {
            printf("%d\n", i - n + 1);
            j = ne[j];
        }
    }
    for (int i = 1; i <= n; i++)
        printf("%d%c", ne[i], i == n ? '\n' : ' ');
    return 0;
}
```

#### Trie

**作用**：维护一个字符串集合，支持插入字符串和查询某个字符串出现了多少次。

**输入**：第一行输入操作数 `n`。接下来 `n` 行，每行输入一个操作和一个字符串：`I str` 表示插入字符串，其他操作表示查询字符串出现次数。

**输出**：对于每次查询，输出该字符串在集合中的出现次数。

**相关知识**：  
Trie 又叫字典树，常用于维护大量字符串。  
每个节点表示一个前缀，`son[p][c]` 表示节点 `p` 经过字符 `c` 转移到的下一个节点。  
`cnt[p]` 表示以节点 `p` 结尾的字符串出现次数。  
该板子默认字符串只含小写字母 `'a' ~ 'z'`。

插入和查询的复杂度都是 `O(|str|)`。

```cpp
#include <bits/stdc++.h>
using namespace std;
const int N = 100010;
int n;
int son[N][26], cnt[N], idx; // son 为子节点编号，cnt 为单词结尾次数，idx 为当前节点编号
char str[N];
void insert(char *str) {
    int p = 0;
    for (int i = 0; str[i]; i++) {
        int u = str[i] - 'a';
        if (!son[p][u]) son[p][u] = ++idx;
        p = son[p][u];
    }
    cnt[p]++;
}
int query(char *str) {
    int p = 0;
    for (int i = 0; str[i]; i++) {
        int u = str[i] - 'a';
        if (!son[p][u]) return 0;
        p = son[p][u];
    }
    return cnt[p];
}
int main() {
    cin >> n;
    while (n--) {
        char op[2];
        scanf("%s%s", op, str);
        if (op[0] == 'I') insert(str);
        else printf("%d\n", query(str));
    }
    return 0;
}
```

### 并查集
**基本功能**
1. 将两个集合合并
2. 询问两个元素是否在一个集合当中

**基本原理**
每个集合用一棵**树**来表示。树根的编号就是整个集合的编号。每个节点存储它的 parent 节点，`p[x]` 表示 `x` 的 parent 节点。

`find(x)` 返回的是 x 所在的集合（根节点）编号

```cpp
int find(int x){
	if(p[x]!=x) return p[x]=find(p[x]);
	return x;
}
```

**三个问题**

- 问题1：如何判断树根
```cpp
if (p[x] == x) 则 x 是树根
```

- 问题2：如何求 x 的集合编号
```cpp
px = find(x)
```
接着询问两个元素是否在一个集合当中
```cpp
if (find(x) == find(y)) 则在一个集合中
```

- 问题3：如何合并两个集合
```cpp
p[find(x)] = find(y) 
```

并查集板子

```cpp
const int N=10010;
int n,m;
int p[N];

int find(int k){
	if(p[k]==k) return p[k]=find(p[k]);
	return k;
}

int main(){
	cin>>n>>m;
	for(int i=1;i<=n;i++) p[i]=i;
	for(int i=1;i<=m;i++){
		int x,y,z;
		cin>>z>>x>>y;
		if(z==1) 				// 合并xy
			p[find(x)]=find(y);
		else {					// 查询xy是否在同一个集合
			if(find(x)==find(y)) 
				cout<<"Y"<<endl; // 在一个集合
			else 
				cout<<"N"<<endl; // 不在一个集合
		}
	}
}
```

#### 维护额外信息：求集合中点的个数
> 在无向图中，可以把连通块看成集合，在输入时不断加边的过程就是在合并集合，**求连通块的个数**也就是求集合个数，**求每个连通块内点的个数**就是求集合中点的个数。

需要增加一个数组 `siz[i]` 来维护每个集合中点的个数， 并且只有 `i==根节点` 的时候，`siz[i]` 才有意义。维护的步骤仅需要在合并集合的同时进行
```cpp
siz[find(b)] += siz[find(a)]
p[find(a)] = find(b)
```
有的时候题目会给重边，这时候在合并之前要判断一下a、b 是不是本来就在一个集合中。

####  维护复杂关系：带权与扩展域
##### 带权并查集

以经典的[【NOI2001】“食物链”问题](https://www.luogu.com.cn/problem/P2024) 为例。题目中有 `n` 个动物，它们属于三类之一，并且三类之间构成循环捕食关系：A 吃 B，B 吃 C，C 吃 A。接下来会给出 `k` 句话，每句话会给出 `op` `x` `y` ，有两种形式：
- `op==1` “x 和 y 是同类”。
- `op==2` “x 吃 y”。

但是这些话不一定都是真的。我们需要按照输入顺序判断每句话是否与前面已经确认的信息矛盾，如果矛盾，就把它计为假话。最后**输出假话的总数**。

题目难点不在于判断单独一句话，而在于“关系会不断累积”。例如已经知道 x 和 y 同类，又知道 y 吃 z，那么之后再出现 x 吃 z 就应该被判断为真；如果出现 z 吃 x，就应该被判断为假。这说明我们需要维护的不只是两个点是否在同一个集合中，还要维护它们之间的相对关系。

**分析**

普通并查集只能回答一个问题：两个元素是否属于同一个集合。它适合维护“连通性”或者“等价关系”，比如判断两个人是否在同一个朋友圈中。但是这道题里，元素之间不只是“在不在同一组”，还有更具体的关系：

```
x 和 y 同类
x 吃 y
x 被 y 吃
```

因此需要使用**带边权的并查集**。

带边权并查集的核心思想是：**在维护父节点 p[x] 的同时，再额外维护一个权值 d[x]，表示 x 和父节点 p[x] 之间的关系。经过路径压缩后，d[x] 就可以表示 x 和所在集合根节点之间的关系。**

在这道题中，因为食物链关系是三类循环，所以可以用 mod 3 来表示关系。

也就是说，如果 d[x] 表示 x 到根节点的关系，那么对于同一个集合中的两个点 x 和 y，可以通过这个公式判断 x 和 y 的关系：

```text
(d[x] - d[y] + 3) % 3 = k
```
那么我们就可以规定结果 k 及其表示：

```
k=0：x 和 y 是同类
k=1：x 吃 y
k=2：x 被 y 吃
```

每次读入一句话时，先判断 x、y 是否越界。如果越界，这句话一定是假话。然后找出 x 和 y 的根节点。

如果 x 和 y 已经在同一个集合中，说明它们之间的关系已经可以由前面的信息推出。此时只需要检查当前这句话是否和已有关系矛盾。

如果 x 和 y 不在同一个集合中，说明这句话提供了新的信息。此时不能简单合并集合，还必须计算出两个集合根节点之间的关系，也就是维护好新的 d[px]，让合并之后整个集合内部的相对关系仍然成立。

这个时候，我们默认选择把x合并到y上，有了如下示意图

```
        d[px]
    px --------> py
    ^            ^
    |            |    
d[x]|            |d[y]
    |            |
    x            y
   ```

在模3的意义下，将式子 `d[x]+d[px]-d[y]= k` 移项之后，我们需要维护的 `d[px]` 就是 `d[px] = d[y] - d[x] + k`，修改为在 mod 3 的意义下就变成了：
```cpp
d[px]=(d[y]-d[x]+k+3)%3;
```




find 函数也要维护 d[x] ，以下这两种写法都可以。

第一种：
先 find(p[x])，但暂时不改 p[x]。
所以 p[x] 仍然指向旧父节点，可以直接用 d[p[x]]。
```cpp
int find(int x){
    if(p[x]!=x){
        int u=find(p[x]);
        d[x]=(d[x]+d[p[x]])%3;        
        p[x]=u;
    }
    return p[x];
}
```


第二种：
先保存旧父节点 u。
然后可以直接改 p[x]。
最后用 d[u] 找到旧父节点的关系。

```cpp
int find(int x){
    if(p[x]!=x){
        int u=p[x];
        p[x]=find(p[x]);
        d[x]=(d[x]+d[u])%3;
    }
    return p[x];
}
```


核心公式永远是：
```
d[x] = (d[x] + d[旧父节点]) % 3;
```
第一种写法里，旧父节点还在 p[x] 里，u 存的是根节点；第二种写法里，旧父节点被提前保存到 u 里。个人感觉还是第二种更易于理解一些，并且和朴素并查集板子的关联更紧密一些。


**完整代码**

```cpp
#include <iostream>
#include <cstring>
#include <algorithm>

using namespace std;
const int N = 1e5+10;
int n, k, cnt;
int p[N], d[N];
/*
mod 3 意义下
dx-dy 余 1 表示 x 吃 y
dx-dy 余 2 表示 y 吃 x
dx-dy 余 0 表示 x 与 y 是同类
*/

int find(int x){
    if(p[x]!=x){
        int u=p[x];
        p[x]=find(p[x]);
        d[x]=(d[x]+d[u])%3;
    }
    return p[x];
}

int main(){
    scanf("%d%d", &n, &k);
    for(int i=1; i<=n; i++) p[i]=i;
    for(int i=1; i<=k; i++){
        int op,x,y;
        scanf("%d%d%d",&op,&x,&y);
        if(x>n||y>n){
            cnt++; continue;
        }
        if(x==y && op==2){
            cnt++; continue;
        }
        
        int px=find(x), py=find(y);
        
        if(px==py){          // 已经在一个集合 此时需要判断真假
            if(op==1)        // x y应该是同类
                if((d[x]-d[y]+3) %3 !=0) cnt++;
            else             // x 应该吃 y
                if((d[x]-d[y]+3) %3 !=1) cnt++; 
        }
        else{                // 不在一个集合 需要合并集合、维护d
            p[px]=py;        // 把x的根节点接到y的根节点上
            if(op==1)        // x y 是同类
                d[px]=(d[y]-d[x]+3)%3;
            else             // x吃y 
                d[px]=(d[y]-d[x]+1+3)%3;
        }
    }
    printf("%d", cnt);
    return 0;
}
```


##### 扩展域并查集
普通并查集只能维护 x 和 y 是否属于同一类，但是很多题不只是“同类关系”，还会有“敌人关系”“捕食关系”“相差关系”等。扩展域并查集，**就是把“一个元素”拆成多个“虚拟元素”**，然后用普通并查集维护这些状态之间的等价关系。

还是以刚刚的食物链为例。我们可以把每个动物 x 拆成 3 个点：

```
x          表示 x 所属的种类
x + n      表示 x 吃的种类
x + 2n     表示吃 x 的种类
```

也可以理解为：

```
x 的 A 域（同类域）：和 x 同类
x 的 B 域（捕食域）：被 x 吃
x 的 C 域（天敌域）：吃 x
```

这样原来有 n 个点，现在变成了 3n 个点。

例如，如果一句话说：

```
x 和 y 是同类
```

那就要合并三组关系：

```cpp
merge(x, y);						// x 的同类 = y 的同类
merge(x + n, y + n);				// x 吃的 = y 吃的
merge(x + 2 * n, y + 2 * n);		// 吃 x 的 = 吃 y 的
```

如果一句话说：

```
x 吃 y
```

那么应该是：

```cpp
merge(x, y + 2 * n);				// x 的同类 = 吃 y 的种类
merge(x + n, y);					// x 吃的种类 = y 的同类
merge(x + 2 * n, y + n);			// 吃 x 的种类 = y 吃的种类
```

当然，不同人的域定义可能不同，所以合并写法也可能换一种。但是核心思想不变：把复杂关系转换成若干个“相等关系”。

> 扩展域并查集常见于两类问题。
> 第一类是二分图、敌人关系。每个点拆成两个：
> ```
> x：x 属于正方阵营
> x + n：x 属于反方阵营
> ```
> 如果 x 和 y 是敌人，就合并：`merge(x, y + n);` `merge(x + n, y);`
> 如果发现：`find(x) == find(x + n)` 说明 x 既在正方，又在反方，矛盾。
> 第二类是“食物链”这种模 3 关系。每个点拆成三个：
> ```
> x
> x + n
> x + 2n
> ```
> 分别表示同类、捕食对象、天敌。




**完整代码**

```cpp
#include <bits/stdc++.h>
using namespace std;
int p[200000];
int n,m,op,x,y,ans;
int find(int x){
    if(x==p[x]) return x;
    return p[x]=find(p[x]);
}
void merge(int x,int y){
    p[find(x)]=find(y);
}
int main(){
    scanf("%d%d",&n,&m);
    for(int i=1; i<=3*n; i++) p[i]=i;		// 初始化的时候要3n的空间
    for(int i=1; i<=m; i++){
        scanf("%d%d%d",&op,&x,&y);
        if(x>n || y>n) ans++;
        else if(op==1){		// x y是同类
            if(find(x)==find(y+n) || find(x)==find(y+n+n)) ans++;
            else{
                merge(x,y);
                merge(x+n,y+n);
                merge(x+n+n,y+n+n);
            }
        }
        else{				// x 吃 y
            if(x==y || find(x)==find(y) || find(x)==find(y+n)) ans++;
            else{
                merge(x,y+n+n);		//y的捕食域加入x
                merge(x+n,y);		//x的天敌域加入y
                merge(x+n+n,y+n);	//x的捕食域是y的同类域.
            }
        }
    }
    printf("%d\n", ans);
}
```


> 带权与扩展域的区别和联系：扩展域并查集可以看作带权并查集在“关系种类有限”时的一种展开写法；它用空间换**直观性**。带权并查集则直接维护点到根节点的关系值，**空间更省**，也能处理**更一般的权值关系**。




### ST表

**作用**：静态区间查询，支持多次查询区间最大值。

**输入**：第一行输入 `n, m`，表示数组长度和询问次数。接下来输入 `n` 个数。随后输入 `m` 组询问，每组输入 `l, r`。

**输出**：对每次询问，输出区间 `[l, r]` 中的最大值。

**相关知识**：  
ST 表适合处理静态区间最值查询，即数组不修改，只查询。  
预处理复杂度为 `O(n log n)`，单次查询复杂度为 `O(1)`。

`f[i][j]` 表示从位置 `i` 开始，长度为 `2^j` 的区间最大值：
```text
f[i][j] = max(a[i...i+2^j-1])
```
转移为：
```text
f[i][j] = max(f[i][j - 1], f[i + 2^(j - 1)][j - 1])
```
查询 `[l, r]` 时，设 `p = log2(r - l + 1)`，用两个长度为 `2^p` 的区间覆盖：
```text
max(f[l][p], f[r - 2^p + 1][p])
```
```cpp
#include <bits/stdc++.h>
using namespace std;
const int N = 100100;
int n, m, len;
int f[N][40]; // f[i][j] 表示从 i 开始，长度为 2^j 的区间最大值
int lg[N]; // lg[i] 表示 log2(i) 向下取整
int read() {
    int x = 0, f = 1;
    char ch = getchar();
    if (ch == '-') f = -1, ch = getchar();
    while (ch < '0' || ch > '9') ch = getchar();
    while (ch >= '0' && ch <= '9') x = (x << 1) + (x << 3) + (ch ^ 48), ch = getchar();
    return x * f;
}
int main() {
    n = read(), m = read();
    for (int i = 2; i <= n; i++) lg[i] = lg[i >> 1] + 1;
    for (int i = 1; i <= n; i++) f[i][0] = read();
    len = lg[n];
    for (int j = 1; j <= len; j++) {
        for (int i = 1; i + (1 << j) - 1 <= n; i++) {
            f[i][j] = max(f[i][j - 1], f[i + (1 << (j - 1))][j - 1]);
        }
    }
    for (int i = 1; i <= m; i++) {
        int l, r;
        l = read(), r = read();
        int p = lg[r - l + 1];
        cout << max(f[l][p], f[r - (1 << p) + 1][p]) << '\n';
    }
    return 0;
}
```

### 树状数组

**作用**：维护一个数组，支持单点修改和区间求和。

**输入**：第一行输入 `n, m`，表示数组长度和操作次数。接下来输入 `n` 个数。随后输入 `m` 组操作：`1 x y` 表示把第 `x` 个数加上 `y`；`2 x y` 表示查询区间 `[x, y]` 的和。

**输出**：对每次区间查询，输出区间和。

**相关知识**：  
树状数组又叫 Fenwick Tree，常用于维护前缀和。  
`a[i]` 在板子中不是原数组，而是树状数组节点，维护一段区间的信息。

核心函数：

```text
lowbit(x) = x & -x
```

`lowbit(x)` 表示 `x` 的二进制中最低位的 `1` 所代表的值。  
树状数组支持：

```text
单点加：O(log n)
前缀和查询：O(log n)
区间和查询：[l, r] = getsum(r) - getsum(l - 1)
```

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
const int N = 610000;
int n, m;
ll a[N]; // a[i] 是树状数组节点，不是原数组
int lowbit(int x) {
    return x & (-x);
}
// 单点修改：给位置 x 加上 k
void update(int x, ll k) {
    for (int i = x; i <= n; i += lowbit(i)) a[i] += k;
}
// 查询前缀和：返回 1 ~ x 的和
ll getsum(int x) {
    ll ans = 0;
    for (int i = x; i; i -= lowbit(i)) ans += a[i];
    return ans;
}
int main() {
    cin >> n >> m;
    for (int i = 1; i <= n; i++) {
        ll x;
        cin >> x;
        update(i, x);
    }
    for (int i = 1; i <= m; i++) {
        int t, x, y;
        cin >> t >> x >> y;
        if (t == 1) update(x, y);
        else cout << getsum(y) - getsum(x - 1) << '\n';
    }
    return 0;
}
```



# 图论

## 简单图论

### 最短路问题

**作用**：求图中两点之间、单个起点到所有点、或任意两点之间的最短距离。

**常见问题类型**：

```text
单源最短路：
给定起点 s，求 s 到所有点的最短距离。
常用 Dijkstra、Bellman-Ford、SPFA。

多源最短路：
求任意两个点之间的最短距离。
常用 Floyd。

限制边数的最短路：
要求最多经过 k 条边。
常用 Bellman-Ford 的备份数组写法。

负环判断：
判断是否存在权值和为负的环。
常用 Bellman-Ford 或 SPFA。

输出路径：
除了求最短距离，还要还原具体经过哪些点。
常用 pre[] 或 path[][] 记录路径。
```

**算法选择表**：

|算法|适用场景|能否处理负权边|能否判负环|复杂度|
|---|---|--:|--:|--:|
|搜索|小数据、暴力验证、简单图|可以|不常用|通常较高|
|Floyd|任意两点最短路|可以|可以|$O(n^3)$|
|Bellman-Ford|单源最短路、限制边数、负权边|可以|可以|$O(nm)$|
|SPFA|Bellman-Ford 的队列优化|可以|可以|平均 $O(km)$，最坏 $O(nm)$|
|Dijkstra 堆优化|非负边权单源最短路|不可以|不可以|$O(m \log n)$|

**核心概念：松弛操作**

如果当前已经知道 `s -> u` 的最短距离为 `dis[u]`，并且有边 `u -> v`，边权为 `w`，那么可以尝试更新：

```text
dis[v] = min(dis[v], dis[u] + w)
```

如果更新成功，说明经过 `u` 到达 `v` 更优。

---

#### 求最短路

##### 搜索 $O(nm)$

**作用**：用搜索思想求最短路，适合数据范围很小、需要快速写暴力验证的情况。

**输入**：第一行输入 `n, m, s, t`，表示点数、边数、起点、终点。接下来 `m` 行，每行输入一条有向边 `u, v, w`。

**输出**：输出 `s` 到 `t` 的最短距离。若不可达，输出 `-1`。

**相关知识**：  
搜索做法本质是枚举从 `s` 到 `t` 的路径，过程中用当前路径长度 `sum` 更新答案。  
若图中有环，必须用 `vis[]` 防止在一条路径中重复经过同一个点，否则可能死循环。  
这种做法不是常规最短路算法，复杂度不稳定，只适合作为小数据暴力或对拍程序。

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
const int maxn = 1010;
const int maxm = 20010;
const ll inf = 0x3f3f3f3f3f3f3f3f;
int n, m, s, t, ct;
int head[maxn], vis[maxn];
ll ans = inf;
struct node {
    int to, next;
    ll w;
} e[maxm];
void add(int u, int v, ll w) {
    ct++;
    e[ct].to = v;
    e[ct].w = w;
    e[ct].next = head[u];
    head[u] = ct;
}
void dfs(int u, ll sum) {
    if (sum >= ans) return;
    if (u == t) {
        ans = min(ans, sum);
        return;
    }
    vis[u] = 1;
    for (int i = head[u]; i; i = e[i].next) {
        int v = e[i].to;
        if (!vis[v]) dfs(v, sum + e[i].w);
    }
    vis[u] = 0;
}
int main() {
    scanf("%d%d%d%d", &n, &m, &s, &t);
    for (int i = 1; i <= m; i++) {
        int u, v;
        ll w;
        scanf("%d%d%lld", &u, &v, &w);
        add(u, v, w);
    }
    dfs(s, 0);
    printf("%lld", ans == inf ? -1 : ans);
    return 0;
}
```

---

##### Floyd $O(n^3)$

**作用**：求任意两点之间的最短路。

**输入**：第一行输入 `n, m`，表示点数和边数。接下来 `m` 行，每行输入 `u, v, w`，表示一条从 `u` 到 `v`，边权为 `w` 的有向边。

**输出**：输出任意两点之间的最短路矩阵。若两点不可达，输出 `INF`。

**相关知识**：  
Floyd 是动态规划。  
`dis[i][j]` 表示当前已知的 `i` 到 `j` 的最短距离。  
枚举中转点 `k`，尝试用路径 `i -> k -> j` 更新 `i -> j`：

```text
dis[i][j] = min(dis[i][j], dis[i][k] + dis[k][j])
```

三层循环顺序必须是 `k, i, j`。  
Floyd 可以处理负权边，但如果存在负环，则最短路没有意义。  
若运行后存在 `dis[i][i] < 0`，说明图中存在负环。

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
const int maxn = 510;
const ll inf = 0x3f3f3f3f3f3f3f3f;
int n, m;
ll dis[maxn][maxn];
void init() {
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= n; j++) {
            if (i == j) dis[i][j] = 0;
            else dis[i][j] = inf;
        }
    }
}
void floyd() {
    for (int k = 1; k <= n; k++) {
        for (int i = 1; i <= n; i++) {
            if (dis[i][k] == inf) continue;
            for (int j = 1; j <= n; j++) {
                if (dis[k][j] == inf) continue;
                dis[i][j] = min(dis[i][j], dis[i][k] + dis[k][j]);
            }
        }
    }
}
int main() {
    scanf("%d%d", &n, &m);
    init();
    for (int i = 1; i <= m; i++) {
        int u, v;
        ll w;
        scanf("%d%d%lld", &u, &v, &w);
        dis[u][v] = min(dis[u][v], w);
    }
    floyd();
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= n; j++) {
            if (dis[i][j] == inf) printf("INF%c", j == n ? '\n' : ' ');
            else printf("%lld%c", dis[i][j], j == n ? '\n' : ' ');
        }
    }
    return 0;
}
```

##### Bellman-Ford $O(nm)$

**作用**：求单源最短路，可以处理负权边，也可以判断从起点可达的负环。

**输入**：第一行输入 `n, m, s`，表示点数、边数、起点。接下来 `m` 行，每行输入 `u, v, w`，表示一条从 `u` 到 `v`，边权为 `w` 的有向边。

**输出**：若不存在负环，输出 `s` 到所有点的最短距离；若不可达，输出 `-1`。

**相关知识**：  
在没有负环的情况下，一条最短路最多经过 `n - 1` 条边。  
Bellman-Ford 每一轮扫描所有边，并尝试松弛。做完 `n - 1` 轮后，如果第 `n` 轮仍然可以松弛，说明存在从起点可达的负环。

`flag` 用于提前退出：如果某一轮没有发生任何更新，说明最短路已经稳定。

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
const int maxn = 100010;
const int maxm = 200010;
const ll inf = 0x3f3f3f3f3f3f3f3f;
int n, m, s;
ll dis[maxn];
struct edge {
    int u, v;
    ll w;
} e[maxm];
void bellman_ford(int s) {
    for (int i = 1; i <= n; i++) dis[i] = inf;
    dis[s] = 0;
    for (int j = 1; j <= n - 1; j++) {
        bool flag = false;
        for (int i = 1; i <= m; i++) {
            int u = e[i].u, v = e[i].v;
            if (dis[u] != inf && dis[v] > dis[u] + e[i].w) {
                dis[v] = dis[u] + e[i].w;
                flag = true;
            }
        }
        if (!flag) break;
    }
}
int main() {
    scanf("%d%d%d", &n, &m, &s);
    for (int i = 1; i <= m; i++)
        scanf("%d%d%lld", &e[i].u, &e[i].v, &e[i].w);
    bellman_ford(s);
    for (int i = 1; i <= n; i++)
        printf("%lld%c", dis[i] == inf ? -1 : dis[i], i == n ? '\n' : ' ');
    return 0;
}
```

##### Bellman-Ford：限制边数最短路 $O(km)$

**作用**：求从起点 `s` 到终点 `t`，最多经过 `k` 条边的最短距离。

**输入**：第一行输入 `n, m, k, s, t`，表示点数、边数、最多经过边数、起点、终点。接下来 `m` 行，每行输入 `u, v, w`。

**输出**：输出最多经过 `k` 条边时，`s` 到 `t` 的最短距离。若不可达，输出 `impossible`。

**相关知识**：  
Bellman-Ford 的第 `i` 轮表示“最多经过 `i` 条边”的最短路。  
如果限制最多经过 `k` 条边，只需要做 `k` 轮松弛。

必须使用 `backup[]` 备份上一轮的 `dis[]`。  
否则同一轮中一个点刚被更新，又立刻拿去更新别的点，会导致一轮内经过多条边，从而破坏“最多经过 `k` 条边”的限制。

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
const int maxn = 100010;
const int maxm = 200010;
const ll inf = 0x3f3f3f3f3f3f3f3f;
int n, m, k, s, t;
ll dis[maxn], backup[maxn];
struct edge {
    int u, v;
    ll w;
} e[maxm];
void bellman_ford() {
    for (int i = 1; i <= n; i++) dis[i] = inf;
    dis[s] = 0;
    for (int j = 1; j <= k; j++) {
        memcpy(backup, dis, sizeof dis);
        for (int i = 1; i <= m; i++) {
            int u = e[i].u, v = e[i].v;
            if (backup[u] != inf) dis[v] = min(dis[v], backup[u] + e[i].w);
        }
    }
}
int main() {
    scanf("%d%d%d%d%d", &n, &m, &k, &s, &t);
    for (int i = 1; i <= m; i++)
        scanf("%d%d%lld", &e[i].u, &e[i].v, &e[i].w);
    bellman_ford();
    if (dis[t] > inf / 2) printf("impossible");
    else printf("%lld", dis[t]);
    return 0;
}
```

##### Spfa $O(km)$

**作用**：求单源最短路，可以处理负权边。

**输入**：第一行输入 `n, m, s`，表示点数、边数、起点。接下来 `m` 行，每行输入 `u, v, w`，表示一条从 `u` 到 `v`，边权为 `w` 的有向边。

**输出**：输出 `s` 到每个点的最短距离。若不可达，输出 `-1`。

**相关知识**：  
SPFA 是 Bellman-Ford 的队列优化。  
Bellman-Ford 每次扫描所有边，而 SPFA 只把被更新过的点放入队列，因为只有这些点的出边可能继续更新其他点。

`vis[u]` 表示点 `u` 当前是否在队列中。  
它不是“是否访问过”，而是“是否正在队列中等待处理”。  
点出队时要令 `vis[u] = 0`，这样后面如果它再次被更新，还能重新入队。

SPFA 在普通随机数据上通常较快，但最坏复杂度仍然是 `O(nm)`，可能被卡。若边权非负，通常优先使用 Dijkstra。

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
const int maxn = 100010;
const int maxm = 200010;
const ll inf = 0x3f3f3f3f3f3f3f3f;
int n, m, s, ct;
int head[maxn], vis[maxn];
ll dis[maxn];
struct node {
    int to, next;
    ll w;
} e[maxm];
void add(int u, int v, ll w) {
    ct++;
    e[ct].to = v;
    e[ct].w = w;
    e[ct].next = head[u];
    head[u] = ct;
}
void spfa(int s) {
    queue<int> q;
    for (int i = 1; i <= n; i++) dis[i] = inf, vis[i] = 0;
    dis[s] = 0;
    vis[s] = 1;
    q.push(s);
    while (!q.empty()) {
        int t = q.front();
        q.pop();
        vis[t] = 0;
        for (int i = head[t]; i; i = e[i].next) {
            int v = e[i].to;
            if (dis[v] > dis[t] + e[i].w) {
                dis[v] = dis[t] + e[i].w;
                if (!vis[v]) {
                    vis[v] = 1;
                    q.push(v);
                }
            }
        }
    }
}
int main() {
    scanf("%d%d%d", &n, &m, &s);
    for (int i = 1; i <= m; i++) {
        int u, v;
        ll w;
        scanf("%d%d%lld", &u, &v, &w);
        add(u, v, w);
    }
    spfa(s);
    for (int i = 1; i <= n; i++)
        printf("%lld%c", dis[i] == inf ? -1 : dis[i], i == n ? '\n' : ' ');
    return 0;
}
```

##### Dijkstra 堆优化 $O(m \log n)$

**作用**：求非负边权图中，从起点 `s` 到所有点的最短路。

**输入**：第一行输入 `n, m, s`，表示点数、边数、起点。接下来 `m` 行，每行输入 `u, v, w`，表示一条从 `u` 到 `v`，边权为 `w` 的有向边。

**输出**：输出 `s` 到每个点的最短距离。若不可达，输出 `-1`。

**相关知识**：  
Dijkstra 的核心贪心是：每次选择当前 `dis` 最小且还未确定最短路的点，这个点的最短距离就已经确定。  
这个结论依赖边权非负，所以 Dijkstra 不能处理负权边。

优先队列中可能保存同一个点的多个旧状态，因此弹出队首时要判断 `vis[t]`。  
如果 `vis[t] = 1`，说明这个点的最短路已经确定，直接跳过即可。

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
const int maxn = 100010;
const int maxm = 200010;
const ll inf = 0x3f3f3f3f3f3f3f3f;
int n, m, s, ct;
int head[maxn], vis[maxn];
ll dis[maxn];
struct edge {
    int to, next;
    ll w;
} e[maxm];
struct node {
    int now;
    ll dis;
    bool operator<(const node &x) const {
        return x.dis < dis;
    }
};
priority_queue<node> q;
void add(int u, int v, ll w) {
    ct++;
    e[ct].to = v;
    e[ct].w = w;
    e[ct].next = head[u];
    head[u] = ct;
}
void dj(int s) {
    for (int i = 1; i <= n; i++) dis[i] = inf, vis[i] = 0;
    dis[s] = 0;
    q.push((node){s, 0});
    while (!q.empty()) {
        int t = q.top().now;
        q.pop();
        if (vis[t]) continue;
        vis[t] = 1;
        for (int i = head[t]; i; i = e[i].next) {
            int v = e[i].to;
            if (dis[v] > dis[t] + e[i].w) {
                dis[v] = dis[t] + e[i].w;
                q.push((node){v, dis[v]});
            }
        }
    }
}
int main() {
    scanf("%d%d%d", &n, &m, &s);
    for (int i = 1; i <= m; i++) {
        int u, v;
        ll w;
        scanf("%d%d%lld", &u, &v, &w);
        add(u, v, w);
    }
    dj(s);
    for (int i = 1; i <= n; i++)
        printf("%lld%c", dis[i] == inf ? -1 : dis[i], i == n ? '\n' : ' ');
    return 0;
}
```

#### 判负环

##### Bellman-Ford

**作用**：判断从起点 `s` 可达的范围内是否存在负环。

**输入**：第一行输入 `n, m, s`，表示点数、边数、起点。接下来 `m` 行，每行输入有向边 `u, v, w`。

**输出**：若存在从 `s` 可达的负环，输出 `YES`；否则输出 `NO`。

**相关知识**：  
Bellman-Ford 判负环的依据是：如果经过 `n - 1` 轮松弛后，第 `n` 轮仍然可以松弛成功，则存在负环。

因为没有负环时，最短路最多只需要经过 `n - 1` 条边。  
第 `n` 轮仍然能变短，说明路径中一定出现了重复点，也就是存在环；并且这个环能让路径继续变短，因此它是负环。

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
const int maxn = 100010;
const int maxm = 200010;
const ll inf = 0x3f3f3f3f3f3f3f3f;
int n, m, s;
ll dis[maxn];
struct edge {
    int u, v;
    ll w;
} e[maxm];
bool bellman_ford(int s) {
    for (int i = 1; i <= n; i++) dis[i] = inf;
    dis[s] = 0;
    for (int j = 1; j <= n - 1; j++) {
        for (int i = 1; i <= m; i++) {
            int u = e[i].u, v = e[i].v;
            if (dis[u] != inf && dis[v] > dis[u] + e[i].w)
                dis[v] = dis[u] + e[i].w;
        }
    }
    for (int i = 1; i <= m; i++) {
        int u = e[i].u, v = e[i].v;
        if (dis[u] != inf && dis[v] > dis[u] + e[i].w) return true;
    }
    return false;
}
int main() {
    scanf("%d%d%d", &n, &m, &s);
    for (int i = 1; i <= m; i++)
        scanf("%d%d%lld", &e[i].u, &e[i].v, &e[i].w);
    printf(bellman_ford(s) ? "YES" : "NO");
    return 0;
}
```

##### SPFA

**作用**：用 SPFA 判断是否存在负环。

**输入**：第一行输入 `n, m, s`，表示点数、边数、起点。接下来 `m` 行，每行输入有向边 `u, v, w`。

**输出**：若存在从 `s` 可达的负环，输出 `YES`；否则输出 `NO`。

**相关知识**：  
SPFA 判负环常见有两种写法：

```text
1. 统计最短路经过的边数 cnt[v]，若 cnt[v] >= n，则存在负环。
2. 统计入队次数 cnt[v]，若 cnt[v] >= n，则存在负环。
```

下面板子采用第一种写法。  
`cnt[v] = cnt[t] + 1` 表示当前到 `v` 的路径边数。  
如果某个点的最短路路径边数达到 `n`，说明这条路径中至少有一个点重复出现，也就是存在环；又因为它是在松弛成功后出现的，所以存在负环。

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
const int maxn = 100010;
const int maxm = 200010;
const ll inf = 0x3f3f3f3f3f3f3f3f;
int n, m, s, ct;
int head[maxn], vis[maxn], cnt[maxn]; // cnt[i] 表示当前到 i 的路径边数
ll dis[maxn];
struct node {
    int to, next;
    ll w;
} e[maxm];
void add(int u, int v, ll w) {
    ct++;
    e[ct].to = v;
    e[ct].w = w;
    e[ct].next = head[u];
    head[u] = ct;
}
bool spfa(int s) {
    queue<int> q;
    for (int i = 1; i <= n; i++) dis[i] = inf, vis[i] = 0, cnt[i] = 0;
    dis[s] = 0;
    vis[s] = 1;
    q.push(s);
    while (!q.empty()) {
        int t = q.front();
        q.pop();
        vis[t] = 0;
        for (int i = head[t]; i; i = e[i].next) {
            int v = e[i].to;
            if (dis[v] > dis[t] + e[i].w) {
                dis[v] = dis[t] + e[i].w;
                cnt[v] = cnt[t] + 1;
                if (cnt[v] >= n) return true;
                if (!vis[v]) {
                    vis[v] = 1;
                    q.push(v);
                }
            }
        }
    }
    return false;
}
int main() {
    scanf("%d%d%d", &n, &m, &s);
    for (int i = 1; i <= m; i++) {
        int u, v;
        ll w;
        scanf("%d%d%lld", &u, &v, &w);
        add(u, v, w);
    }
    printf(spfa(s) ? "YES" : "NO");
    return 0;
}
```

#### 输出路径

##### Floyd

**作用**：求任意两点之间的最短路，并输出指定起点 `s` 到终点 `t` 的最短路径。

**输入**：第一行输入 `n, m, s, t`，表示点数、边数、起点、终点。接下来 `m` 行，每行输入 `u, v, w`。

**输出**：输出 `s` 到 `t` 的最短路径经过的点。若不可达，输出 `No path`。

**相关知识**：  
`path[i][j]` 表示从 `i` 到 `j` 的最短路径上，`i` 后面的第一个点。  
初始化时，如果存在边 `u -> v`，则令：

```text
path[u][v] = v
```

当 `dis[i][j]` 被 `i -> k -> j` 更新时，说明从 `i` 出发后，应该先沿着 `i -> k` 的最短路走，因此：

```text
path[i][j] = path[i][k]
```

输出路径时，从 `s` 开始，不断令 `s = path[s][t]`，直到走到 `t`。

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
const int maxn = 510;
const ll inf = 0x3f3f3f3f3f3f3f3f;
int n, m, s, t;
ll dis[maxn][maxn];
int path[maxn][maxn]; // path[i][j] 表示 i 到 j 的路径上 i 的下一步
void init() {
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= n; j++) {
            dis[i][j] = inf;
            path[i][j] = 0;
        }
        dis[i][i] = 0;
        path[i][i] = i;
    }
}
void floyd() {
    for (int k = 1; k <= n; k++) {
        for (int i = 1; i <= n; i++) {
            if (dis[i][k] == inf) continue;
            for (int j = 1; j <= n; j++) {
                if (dis[k][j] == inf) continue;
                if (dis[i][j] > dis[i][k] + dis[k][j]) {
                    dis[i][j] = dis[i][k] + dis[k][j];
                    path[i][j] = path[i][k];
                }
            }
        }
    }
}
void print_path() {
    if (dis[s][t] == inf) {
        printf("No path");
        return;
    }
    int u = s;
    printf("%d", u);
    while (u != t) {
        u = path[u][t];
        printf(" %d", u);
    }
}
int main() {
    scanf("%d%d%d%d", &n, &m, &s, &t);
    init();
    for (int i = 1; i <= m; i++) {
        int u, v;
        ll w;
        scanf("%d%d%lld", &u, &v, &w);
        if (dis[u][v] > w) {
            dis[u][v] = w;
            path[u][v] = v;
        }
    }
    floyd();
    print_path();
    return 0;
}
```

##### SPFA

**作用**：求从起点 `s` 到终点 `t` 的最短路，并输出具体路径。

**输入**：第一行输入 `n, m, s, t`，表示点数、边数、起点、终点。接下来 `m` 行，每行输入有向边 `u, v, w`。

**输出**：输出 `s` 到 `t` 的最短距离和路径。若不可达，输出 `No path`。

**相关知识**：  
SPFA 输出路径时，用 `pre[v]` 记录最短路上 `v` 的前驱。  
当通过边 `u -> v` 成功松弛 `v` 时，说明当前最优路径里 `v` 的前一个点变成了 `u`，因此：

```text
pre[v] = u
```

输出路径时，从终点 `t` 沿着 `pre[]` 不断往前跳到起点 `s`，再反转输出。

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
const int maxn = 100010;
const int maxm = 200010;
const ll inf = 0x3f3f3f3f3f3f3f3f;
int n, m, s, t, ct;
int head[maxn], vis[maxn], pre[maxn]; // pre[i] 表示最短路上 i 的前驱
ll dis[maxn];
struct node {
    int to, next;
    ll w;
} e[maxm];
void add(int u, int v, ll w) {
    ct++;
    e[ct].to = v;
    e[ct].w = w;
    e[ct].next = head[u];
    head[u] = ct;
}
void spfa(int s) {
    queue<int> q;
    for (int i = 1; i <= n; i++) dis[i] = inf, vis[i] = 0, pre[i] = 0;
    dis[s] = 0;
    vis[s] = 1;
    q.push(s);
    while (!q.empty()) {
        int u = q.front();
        q.pop();
        vis[u] = 0;
        for (int i = head[u]; i; i = e[i].next) {
            int v = e[i].to;
            if (dis[v] > dis[u] + e[i].w) {
                dis[v] = dis[u] + e[i].w;
                pre[v] = u;
                if (!vis[v]) {
                    vis[v] = 1;
                    q.push(v);
                }
            }
        }
    }
}
void print_path() {
    if (dis[t] == inf) {
        printf("No path");
        return;
    }
    vector<int> path;
    for (int i = t; i; i = pre[i]) path.push_back(i);
    reverse(path.begin(), path.end());
    printf("%lld\n", dis[t]);
    for (int i = 0; i < (int)path.size(); i++)
        printf("%d%c", path[i], i + 1 == (int)path.size() ? '\n' : ' ');
}
int main() {
    scanf("%d%d%d%d", &n, &m, &s, &t);
    for (int i = 1; i <= m; i++) {
        int u, v;
        ll w;
        scanf("%d%d%lld", &u, &v, &w);
        add(u, v, w);
    }
    spfa(s);
    print_path();
    return 0;
}
```

##### Dijkstra

**作用**：求非负边权图中，从起点 `s` 到终点 `t` 的最短路，并输出具体路径。

**输入**：第一行输入 `n, m, s, t`，表示点数、边数、起点、终点。接下来 `m` 行，每行输入有向边 `u, v, w`。

**输出**：输出 `s` 到 `t` 的最短距离和路径。若不可达，输出 `No path`。

**相关知识**：  
Dijkstra 输出路径同样使用 `pre[]`。  
当通过 `u -> v` 成功松弛 `v` 时，记录：

```text
pre[v] = u
```

最后从终点 `t` 倒着沿 `pre[]` 回到起点 `s`，再反转输出。

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
const int maxn = 100010;
const int maxm = 200010;
const ll inf = 0x3f3f3f3f3f3f3f3f;
int n, m, s, t, ct;
int head[maxn], vis[maxn], pre[maxn]; // pre[i] 表示最短路上 i 的前驱
ll dis[maxn];
struct edge {
    int to, next;
    ll w;
} e[maxm];
struct node {
    int now;
    ll dis;
    bool operator<(const node &x) const {
        return x.dis < dis;
    }
};
priority_queue<node> q;
void add(int u, int v, ll w) {
    ct++;
    e[ct].to = v;
    e[ct].w = w;
    e[ct].next = head[u];
    head[u] = ct;
}
void dj(int s) {
    for (int i = 1; i <= n; i++) dis[i] = inf, vis[i] = 0, pre[i] = 0;
    dis[s] = 0;
    q.push((node){s, 0});
    while (!q.empty()) {
        int u = q.top().now;
        q.pop();
        if (vis[u]) continue;
        vis[u] = 1;
        for (int i = head[u]; i; i = e[i].next) {
            int v = e[i].to;
            if (dis[v] > dis[u] + e[i].w) {
                dis[v] = dis[u] + e[i].w;
                pre[v] = u;
                q.push((node){v, dis[v]});
            }
        }
    }
}
void print_path() {
    if (dis[t] == inf) {
        printf("No path");
        return;
    }
    vector<int> path;
    for (int i = t; i; i = pre[i]) path.push_back(i);
    reverse(path.begin(), path.end());
    printf("%lld\n", dis[t]);
    for (int i = 0; i < (int)path.size(); i++)
        printf("%d%c", path[i], i + 1 == (int)path.size() ? '\n' : ' ');
}
int main() {
    scanf("%d%d%d%d", &n, &m, &s, &t);
    for (int i = 1; i <= m; i++) {
        int u, v;
        ll w;
        scanf("%d%d%lld", &u, &v, &w);
        add(u, v, w);
    }
    dj(s);
    print_path();
    return 0;
}
```

### Tarjan
#### Tarjan 缩点

**作用**：求有向图的强连通分量，将每个强连通分量缩成一个点，得到一张 DAG，并在缩点后的 DAG 上求最大点权路径。

**输入**：第一行输入 `n, m`，表示点数和边数。第二行输入每个点的点权 `a[i]`。接下来 `m` 行，每行输入一条有向边 `u -> v`。

**输出**：输出缩点后 DAG 上的最大点权路径和。

**相关知识**：  
强连通分量指有向图中的一个极大点集，点集中任意两个点都可以互相到达。  
Tarjan 中：

```text
dfn[u]：点 u 被第一次访问的时间戳
low[u]：点 u 能追溯到的最早时间戳
vis[u]：点 u 是否还在栈中
be[u]：点 u 所属强连通分量的代表点
w[u]：缩点后，代表点 u 的新点权
```

当 `dfn[u] == low[u]` 时，`u` 是一个强连通分量的根。  
缩点后得到 DAG，可以用拓扑排序做 DP。  
若原图有环，直接 DP 不方便；缩点后变成 DAG，才能按拓扑序转移。

```cpp
#include <bits/stdc++.h>
using namespace std;
const int maxn = 10200;
int n, m, a[maxn], c;
int dfn[maxn], low[maxn], vis[maxn], be[maxn];
int w[maxn], in[maxn], dis[maxn], ans;
vector<int> e[maxn], p[maxn]; // e 为原图，p 为缩点后的 DAG
queue<int> q;
stack<int> s;
void tarjan(int u) {
    dfn[u] = low[u] = ++c;
    s.push(u);
    vis[u] = 1;
    for (int i = 0; i < (int)e[u].size(); i++) {
        int v = e[u][i];
        if (!dfn[v]) {
            tarjan(v);
            low[u] = min(low[u], low[v]);
        } else if (vis[v]) {
            low[u] = min(low[u], dfn[v]);
        }
    }
    if (dfn[u] == low[u]) {
        int top;
        do {
            top = s.top();
            s.pop();
            be[top] = u;
            vis[top] = 0;
        } while (top != u);
    }
}
void build() {
    for (int u = 1; u <= n; u++) {
        w[be[u]] += a[u];
        for (int i = 0; i < (int)e[u].size(); i++) {
            int v = e[u][i];
            if (be[u] == be[v]) continue;
            p[be[u]].push_back(be[v]);
            in[be[v]]++;
        }
    }
}
void topo() {
    for (int i = 1; i <= n; i++)
        if (i == be[i] && in[i] == 0) q.push(i), dis[i] = w[i];
    while (!q.empty()) {
        int u = q.front();
        q.pop();
        for (int i = 0; i < (int)p[u].size(); i++) {
            int v = p[u][i];
            dis[v] = max(dis[v], dis[u] + w[v]);
            in[v]--;
            if (!in[v]) q.push(v);
        }
    }
}
int main() {
    cin >> n >> m;
    for (int i = 1; i <= n; i++) cin >> a[i];
    for (int i = 1; i <= m; i++) {
        int u, v;
        cin >> u >> v;
        e[u].push_back(v);
    }
    for (int i = 1; i <= n; i++)
        if (!dfn[i]) tarjan(i);
    build();
    topo();
    for (int i = 1; i <= n; i++)
        ans = max(ans, dis[i]);
    printf("%d", ans);
    return 0;
}
```

#### Tarjan 求割边 `dfn[u]<low[v]`

**作用**：求无向图中的所有割边。割边也叫桥，删除这条边后，图的连通块数量会增加。

**输入**：第一行输入 `n, m`，表示点数和边数。接下来 `m` 行，每行输入一条无向边 `a, b`。

**输出**：按编号升序输出所有割边，每行输出一条边的两个端点，较小编号在前。

**相关知识**：  
在 DFS 树中，若边 `u -> v` 是树边，并且满足：

```text
dfn[u] < low[v]
```

则边 `(u, v)` 是割边。

含义是：`v` 及其子树无法通过非树边回到 `u` 或 `u` 的祖先，因此删除 `(u, v)` 后，`v` 子树会与上方断开。

无向图求割边时，不能只用 `v != fa` 跳过父节点，因为重边会出错。更稳的写法是给每条边编号，DFS 时跳过当前边的反向边 `i ^ 1`。

```cpp
#include <bits/stdc++.h>
using namespace std;
const int maxn = 10010;
const int maxm = 200010;
int n, m, c;
int head[maxn];
int dfn[maxn], low[maxn];
int cnt = 1, act;
struct edge {
    int x, y;
} ans[maxm];
struct node {
    int to, next, from;
} e[maxm];
void add(int u, int v) {
    cnt++;
    e[cnt].to = v;
    e[cnt].from = u;
    e[cnt].next = head[u];
    head[u] = cnt;
}
void add_br(int u, int v) {
    if (u > v) swap(u, v);
    act++;
    ans[act].x = u;
    ans[act].y = v;
}
bool cmp(edge a, edge b) {
    if (a.x == b.x) return a.y < b.y;
    return a.x < b.x;
}
int read() {
    int x = 0, f = 1;
    char ch = getchar();
    if (ch == '-') f = -1, ch = getchar();
    while (ch < '0' || ch > '9') ch = getchar();
    while (ch >= '0' && ch <= '9') x = (x << 1) + (x << 3) + (ch ^ 48), ch = getchar();
    return x * f;
}
void tarjan(int u, int in_edge) {
    dfn[u] = low[u] = ++c;
    for (int i = head[u]; i; i = e[i].next) {
        int v = e[i].to;
        if (!dfn[v]) {
            tarjan(v, i);
            low[u] = min(low[u], low[v]);
            if (dfn[u] < low[v]) add_br(u, v);
        } else if (i != (in_edge ^ 1)) {
            low[u] = min(low[u], dfn[v]);
        }
    }
}
int main() {
    n = read(), m = read();
    for (int i = 1; i <= m; i++) {
        int a = read(), b = read();
        add(a, b);
        add(b, a);
    }
    for (int i = 1; i <= n; i++)
        if (!dfn[i]) tarjan(i, 0);
    sort(ans + 1, ans + 1 + act, cmp);
    for (int i = 1; i <= act; i++)
        printf("%d %d\n", ans[i].x, ans[i].y);
    return 0;
}
```

#### Tarjan 求割点 `dfn[u]<low[v]`

**作用**：求无向图中的所有割点。割点指删除该点后，图的连通块数量会增加的点。

**输入**：第一行输入 `n, m`，表示点数和边数。接下来 `m` 行，每行输入一条无向边 `u, v`。

**输出**：第一行输出割点数量，第二行按编号升序输出所有割点。

**相关知识**：  
对于非根节点 `u`，若存在一个子节点 `v`，满足：

```text
dfn[u] <= low[v]
```

则 `u` 是割点。

含义是：`v` 子树不能绕过 `u` 回到 `u` 的祖先，因此删除 `u` 后，`v` 子树会与上方断开。

对于 DFS 树根节点，判断规则不同：根节点必须至少有两个 DFS 子树，才是割点。

```cpp
#include <bits/stdc++.h>
using namespace std;
const int maxn = 100010;
int n, m, c, cnt;
int dfn[maxn], low[maxn], cut[maxn], root;
vector<int> e[maxn];
void add(int u, int v) {
    e[u].push_back(v);
    e[v].push_back(u);
}
void tarjan(int u) {
    int flag = 0; // flag 记录满足割点条件的子树数量
    dfn[u] = low[u] = ++c;
    for (int i = 0; i < (int)e[u].size(); i++) {
        int v = e[u][i];
        if (!dfn[v]) {
            tarjan(v);
            low[u] = min(low[u], low[v]);
            if (dfn[u] <= low[v]) {
                flag++;
                if (flag > 1 || u != root) cut[u] = 1;
            }
        } else {
            low[u] = min(low[u], dfn[v]);
        }
    }
}
int main() {
    scanf("%d%d", &n, &m);
    for (int i = 1; i <= m; i++) {
        int u, v;
        scanf("%d%d", &u, &v);
        add(u, v);
    }
    for (int i = 1; i <= n; i++)
        if (!dfn[i]) root = i, tarjan(i);
    for (int i = 1; i <= n; i++)
        if (cut[i]) cnt++;
    printf("%d\n", cnt);
    for (int i = 1; i <= n; i++)
        if (cut[i]) printf("%d ", i);
    return 0;
}
```
### SPFA 差分约束 && 判负环

**作用**：用 SPFA 求差分约束系统的一组可行解，并判断是否存在负环。若存在负环，则差分约束系统无解。

**输入**：第一行输入 `n, m`，表示变量个数和约束条数。接下来 `m` 行，每行输入 `u, v, w`，表示一个约束：

```text
x[u] - x[v] <= w
```

**输出**：若无解，输出 `NO`。若有解，输出一组可行解 `dis[1...n]`。

**相关知识**：  
差分约束的标准转化：

```text
x[u] - x[v] <= w
等价于
x[u] <= x[v] + w
建边 v -> u，边权为 w
```

然后问题转化为最短路约束。若图中存在负环，则约束系统无解。

为了保证所有点都能被访问到，需要建立超级源点 `0`，并连边：

```text
0 -> i，边权为 0
```

判负环常用方法：如果某个点入队次数大于等于总点数，则存在负环。这里加入了超级源点，所以总点数是 `n + 1`。

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
const int maxn = 100010;
const int maxm = 300010;
const ll inf = 0x3f3f3f3f3f3f3f3f;
int n, m, ct;
bool flag;
int head[maxn], vis[maxn], cnt[maxn]; // cnt[i] 表示点 i 的入队次数
ll dis[maxn]; // dis[i] 是差分约束的一组可行解
struct node {
    int to, next;
    ll w;
} e[maxm];
void add(int u, int v, ll w) {
    ct++;
    e[ct].to = v;
    e[ct].w = w;
    e[ct].next = head[u];
    head[u] = ct;
}
void spfa() {
    queue<int> q;
    for (int i = 0; i <= n; i++) dis[i] = inf;
    dis[0] = 0;
    q.push(0);
    vis[0] = 1;
    cnt[0] = 1;
    while (!q.empty()) {
        int t = q.front();
        q.pop();
        vis[t] = 0;
        for (int i = head[t]; i; i = e[i].next) {
            int v = e[i].to;
            if (dis[v] > dis[t] + e[i].w) {
                dis[v] = dis[t] + e[i].w;
                if (!vis[v]) {
                    vis[v] = 1;
                    cnt[v]++;
                    if (cnt[v] >= n + 1) {
                        flag = true;
                        return;
                    }
                    q.push(v);
                }
            }
        }
    }
}
int main() {
    scanf("%d%d", &n, &m);
    for (int i = 1; i <= m; i++) {
        int u, v;
        ll w;
        scanf("%d%d%lld", &u, &v, &w);
        add(v, u, w);
    }
    for (int i = 1; i <= n; i++)
        add(0, i, 0);
    spfa();
    if (flag) {
        printf("NO");
        return 0;
    }
    for (int i = 1; i <= n; i++)
        printf("%lld%c", dis[i], i == n ? '\n' : ' ');
    return 0;
}
``` 

### LCA

**作用**：在一棵树上，快速求两个点的最近公共祖先。

**输入**：第一行输入 `n, m, root`，表示点数、询问次数、根节点。接下来 `n - 1` 行输入树边。随后 `m` 行，每行输入两个点 `u, v`。

**输出**：对每次询问，输出 `u` 和 `v` 的最近公共祖先。

**相关知识**：  
倍增 LCA 的核心是预处理每个点的 `2^i` 级祖先：

```text
f[u][i] = u 的 2^i 级祖先
```

转移为：

```text
f[u][i] = f[f[u][i - 1]][i - 1]
```

查询时先把较深的点向上跳到同一深度，再让两个点一起向上跳，直到它们的父亲相同。  
预处理复杂度为 `O(n log n)`，单次查询复杂度为 `O(log n)`。

```cpp
#include <bits/stdc++.h>
using namespace std;
const int N = 500005;
int n, m, root, ct;
int head[N];
int f[N][30], dep[N];
struct node {
    int to, next;
} e[N << 1];
void add(int u, int v) {
    ct++;
    e[ct].to = v;
    e[ct].next = head[u];
    head[u] = ct;
}
void dfs(int p, int fa) {
    f[p][0] = fa;
    dep[p] = dep[fa] + 1;
    for (int i = 1; i < 30; i++)
        f[p][i] = f[f[p][i - 1]][i - 1];
    for (int i = head[p]; i; i = e[i].next)
        if (e[i].to != fa) dfs(e[i].to, p);
}
int lca(int u, int v) {
    if (dep[u] < dep[v]) swap(u, v);
    for (int i = 29; i >= 0; i--)
        if (dep[f[u][i]] >= dep[v]) u = f[u][i];
    if (u == v) return u;
    for (int i = 29; i >= 0; i--)
        if (f[u][i] != f[v][i]) u = f[u][i], v = f[v][i];
    return f[u][0];
}
int main() {
    scanf("%d%d%d", &n, &m, &root);
    for (int i = 1; i < n; i++) {
        int u, v;
        scanf("%d%d", &u, &v);
        add(u, v);
        add(v, u);
    }
    dfs(root, 0);
    for (int i = 1; i <= m; i++) {
        int u, v;
        scanf("%d%d", &u, &v);
        printf("%d\n", lca(u, v));
    }
    return 0;
}
```

#### LCA + 树上差分

**作用**：给定树上若干条路径，统计每个点被多少条路径经过，并求最大经过次数。

**输入**：第一行输入 `n, k`，表示点数和路径条数。接下来 `n - 1` 行输入树边。随后 `k` 行，每行输入一条路径的两个端点 `u, v`。

**输出**：输出所有点中，被路径经过次数的最大值。

**相关知识**：  
树上点差分用于快速统计多条路径对点的覆盖次数。  
对于一条路径 `u -> v`，设 `t = lca(u, v)`，点差分写法为：

```text
cf[u]++
cf[v]++
cf[t]--
cf[fa[t]]--
```

之后从叶子向根回收差分值：

```text
cf[u] += Σ cf[v]，其中 v 是 u 的儿子
```

最终 `cf[u]` 就表示点 `u` 被多少条路径经过。  
注意：如果统计的是边被经过次数，常用写法是 `cf[u]++, cf[v]++, cf[t] -= 2`，与点差分不同。

```cpp
#include <bits/stdc++.h>
using namespace std;
const int N = 101000;
int n, k, ct, ans;
int f[N][35], dep[N];
int cf[N], head[N]; // cf 为树上差分数组
struct node {
    int to, next;
} e[N << 1];
void add(int u, int v) {
    ct++;
    e[ct].to = v;
    e[ct].next = head[u];
    head[u] = ct;
}
void dfs(int u, int fa) {
    f[u][0] = fa;
    dep[u] = dep[fa] + 1;
    for (int i = 1; i < 35; i++)
        f[u][i] = f[f[u][i - 1]][i - 1];
    for (int i = head[u]; i; i = e[i].next)
        if (e[i].to != fa) dfs(e[i].to, u);
}
int lca(int u, int v) {
    if (dep[u] < dep[v]) swap(u, v);
    for (int i = 34; i >= 0; i--)
        if (dep[f[u][i]] >= dep[v]) u = f[u][i];
    if (u == v) return u;
    for (int i = 34; i >= 0; i--)
        if (f[u][i] != f[v][i]) u = f[u][i], v = f[v][i];
    return f[u][0];
}
void get(int u, int fa) {
    for (int i = head[u]; i; i = e[i].next) {
        int v = e[i].to;
        if (v == fa) continue;
        get(v, u);
        cf[u] += cf[v];
    }
    ans = max(ans, cf[u]);
}
int main() {
    cin >> n >> k;
    for (int i = 1; i < n; i++) {
        int u, v;
        cin >> u >> v;
        add(u, v);
        add(v, u);
    }
    dfs(1, 0);
    for (int i = 1; i <= k; i++) {
        int u, v;
        cin >> u >> v;
        int t = lca(u, v);
        cf[u]++;
        cf[v]++;
        cf[t]--;
        cf[f[t][0]]--;
    }
    get(1, 0);
    printf("%d", ans);
    return 0;
}
```

### 欧拉路

**作用**：求有向图的一条欧拉路径，即经过每条边恰好一次的路径。

**输入**：第一行输入 `n, m`，表示点数和边数。接下来 `m` 行，每行输入一条有向边 `u -> v`。

**输出**：若存在欧拉路径，输出路径经过的点序列；否则输出 `No`。

**相关知识**：  
欧拉路径和欧拉回路的判定：

```text
有向图欧拉路径：
存在 1 个点满足 out = in + 1，作为起点；
存在 1 个点满足 in = out + 1，作为终点；
其余点 in = out。
或者所有点 in = out，此时存在欧拉回路。

有向图欧拉回路：
所有点 in = out。

无向图欧拉路径：
恰好有 2 个点度数为奇数，其余点度数为偶数；
或者所有点度数为偶数，此时存在欧拉回路。

无向图欧拉回路：
所有点度数均为偶数。
```

Hierholzer 算法用于构造欧拉路径。  
`del[x]` 表示当前点 `x` 已经遍历到第几条出边，避免重复扫描。  
DFS 回溯时把点加入答案，最后需要倒序输出。  
如果最终路径点数不是 `m + 1`，说明存在边没有被访问到，欧拉路径不存在。

```cpp
#include <bits/stdc++.h>
using namespace std;
const int maxn = 100010;
int n, m;
int st, ed, ctst, cted;
int in[maxn], ou[maxn], ans[maxn], del[maxn];
vector<int> e[maxn];
void dfs(int x) {
    for (int i = del[x]; i < (int)e[x].size(); i = del[x]) {
        del[x] = i + 1;
        int v = e[x][i];
        dfs(v);
    }
    ans[++ans[0]] = x;
}
int main() {
    scanf("%d%d", &n, &m);
    for (int i = 1; i <= m; i++) {
        int u, v;
        scanf("%d%d", &u, &v);
        e[u].push_back(v);
        in[v]++;
        ou[u]++;
    }
    for (int i = 1; i <= n; i++)
        sort(e[i].begin(), e[i].end());
    bool flag = true;
    for (int i = 1; i <= n; i++) {
        if (ou[i] - in[i] == 1) st = i, ctst++;
        else if (in[i] - ou[i] == 1) ed = i, cted++;
        else if (in[i] != ou[i]) flag = false;
    }
    if (ctst > 1 || cted > 1 || ctst != cted) flag = false;
    if (!flag) {
        printf("No");
        return 0;
    }
    if (!st) {
        for (int i = 1; i <= n; i++) {
            if (ou[i]) {
                st = i;
                break;
            }
        }
        if (!st) st = 1;
    }
    dfs(st);
    if (ans[0] != m + 1) {
        printf("No");
        return 0;
    }
    for (int i = ans[0]; i >= 1; i--)
        printf("%d%c", ans[i], i == 1 ? '\n' : ' ');
    return 0;
}
```



### Toposort
#### 拓扑排序 Toposort：保证有解版本（吴欢）

**作用**：对有向无环图进行拓扑排序，输出一种合法拓扑序。

**输入**：第一行输入 `n, m`，表示点数和边数。接下来 `m` 行，每行输入一条有向边 `u, v`，表示 `u` 必须排在 `v` 前面。

**输出**：输出一种拓扑序。

**相关知识**：  
该板子适用于题目保证图中无环、一定存在拓扑序的情况。  
拓扑排序的核心做法是：先把所有入度为 `0` 的点加入队列，每次取出队首点加入答案，并删除它的所有出边。  
由于该版本默认一定有解，所以不需要判断 `c == n`。

变量含义：

```text
head[u]：点 u 的第一条出边编号
f[i]：点 i 的入度
qq[i]：拓扑序中的第 i 个点
c：当前已经加入拓扑序的点数
cnt：当前边编号
```

```cpp
#include <bits/stdc++.h>
using namespace std;
const int maxn = 100010;
const int maxm = 200010;
int n, m, cnt, c;
int head[maxn];
int f[maxn], qq[maxn]; // f 为入度，qq 记录拓扑序
struct node {
    int to, next;
} e[maxm];
void add(int u, int v) {
    cnt++;
    e[cnt].to = v;
    e[cnt].next = head[u];
    head[u] = cnt;
}
void topo() {
    queue<int> q;
    for (int i = 1; i <= n; i++)
        if (f[i] == 0) q.push(i);
    while (!q.empty()) {
        int t = q.front();
        q.pop();
        qq[++c] = t;
        for (int i = head[t]; i; i = e[i].next) {
            int v = e[i].to;
            f[v]--;
            if (f[v] == 0) q.push(v);
        }
    }
}
int main() {
    scanf("%d%d", &n, &m);
    for (int i = 1; i <= m; i++) {
        int u, v;
        scanf("%d%d", &u, &v);
        add(u, v);
        f[v]++;
    }
    topo();
    for (int i = 1; i <= n; i++)
        printf("%d%c", qq[i], i == n ? '\n' : ' ');
    return 0;
}
```

#### 拓扑排序 Toposort：排序并判环

**作用**：对有向图进行拓扑排序，得到一个满足所有先后关系的点序列，并判断图中是否存在环。

**输入**：第一行输入 `n, m`，表示点数和边数。接下来 `m` 行，每行输入一条有向边 `u, v`，表示 `u` 必须排在 `v` 前面。

**输出**：若图中不存在环，输出一种拓扑序；若图中存在环，输出 `No`。

**相关知识**：  
拓扑排序只适用于有向图。若一个有向图存在合法拓扑序，则它一定是 DAG，即有向无环图。  
核心做法是：每次取出一个入度为 `0` 的点，将它加入答案，并删除它的所有出边。若最后加入答案的点数不足 `n`，说明图中存在环。

变量含义：

```text
head[u]：点 u 的第一条出边编号
f[i]：点 i 的原始入度
in[i]：点 i 的临时入度，拓扑排序过程中会被修改
qq[i]：拓扑序中的第 i 个点
c：当前已经加入拓扑序的点数
cnt：当前边编号
```

如果需要判断拓扑序是否唯一，可以在每一步检查队列大小：若某一时刻入度为 `0` 的点超过一个，则拓扑序不唯一。

```cpp
#include <bits/stdc++.h>
using namespace std;
const int maxn = 100010;
const int maxm = 200010;
int n, m, cnt, c;
int head[maxn];
int f[maxn], in[maxn], qq[maxn]; // f 为原始入度，in 为临时入度，qq 记录拓扑序
struct node {
    int to, next;
} e[maxm];
void add(int u, int v) {
    cnt++;
    e[cnt].to = v;
    e[cnt].next = head[u];
    head[u] = cnt;
}
bool topo() {
    queue<int> q;
    c = 0;
    for (int i = 1; i <= n; i++) {
        in[i] = f[i];
        if (in[i] == 0) q.push(i);
    }
    while (!q.empty()) {
        int t = q.front();
        q.pop();
        qq[++c] = t;
        for (int i = head[t]; i; i = e[i].next) {
            int v = e[i].to;
            in[v]--;
            if (in[v] == 0) q.push(v);
        }
    }
    return c == n;
}
int main() {
    scanf("%d%d", &n, &m);
    for (int i = 1; i <= m; i++) {
        int u, v;
        scanf("%d%d", &u, &v);
        add(u, v);
        f[v]++;
    }
    if (!topo()) {
        printf("No");
        return 0;
    }
    for (int i = 1; i <= n; i++)
        printf("%d%c", qq[i], i == n ? '\n' : ' ');
    return 0;
}
```

#### 拓扑排序：判断拓扑序是否唯一

**作用**：在拓扑排序的同时，判断有向图是否有环，以及拓扑序是否唯一。

**输入**：第一行输入 `n, m`，表示点数和边数。接下来 `m` 行，每行输入一条有向边 `u, v`，表示 `u` 必须排在 `v` 前面。

**输出**：若存在环，输出 `No`；若不存在环但拓扑序不唯一，输出 `Not unique`；若拓扑序唯一，输出该拓扑序。

**相关知识**：  
拓扑排序过程中，若某一时刻队列中有多个入度为 `0` 的点，说明这些点之间没有确定的先后关系，因此拓扑序不唯一。  
若整个过程结束后 `c < n`，说明还有点没有进入拓扑序，即图中存在环。

```cpp
#include <bits/stdc++.h>
using namespace std;
const int maxn = 100010;
const int maxm = 200010;
int n, m, cnt, c;
int head[maxn];
int f[maxn], in[maxn], qq[maxn]; // f 为原始入度，in 为临时入度，qq 记录拓扑序
bool flag = true; // flag 表示拓扑序是否唯一
struct node {
    int to, next;
} e[maxm];
void add(int u, int v) {
    cnt++;
    e[cnt].to = v;
    e[cnt].next = head[u];
    head[u] = cnt;
}
bool topo() {
    queue<int> q;
    c = 0;
    flag = true;
    for (int i = 1; i <= n; i++) {
        in[i] = f[i];
        if (in[i] == 0) q.push(i);
    }
    while (!q.empty()) {
        if ((int)q.size() > 1) flag = false;
        int t = q.front();
        q.pop();
        qq[++c] = t;
        for (int i = head[t]; i; i = e[i].next) {
            int v = e[i].to;
            in[v]--;
            if (in[v] == 0) q.push(v);
        }
    }
    return c == n;
}
int main() {
    scanf("%d%d", &n, &m);
    for (int i = 1; i <= m; i++) {
        int u, v;
        scanf("%d%d", &u, &v);
        add(u, v);
        f[v]++;
    }
    if (!topo()) {
        printf("No");
        return 0;
    }
    if (!flag) {
        printf("Not unique");
        return 0;
    }
    for (int i = 1; i <= n; i++)
        printf("%d%c", qq[i], i == n ? '\n' : ' ');
    return 0;
}
```

```cpp
#include<iostream>

#include<queue>
#include<cstring>

using namespace std;
const int N=11000;
const int M=210000;

int n,m,s,t,cnt=1;
int head[N],now[N],dep[N]; 

struct node{
	int to,next,w;
}e[M];

int read(){
	int x=0,f=1; char ch=getchar();
	if(ch=='-') f=-1,ch=getchar();
	while(ch<'0'||ch>'9') ch=getchar();
	while(ch>='0'&&ch<='9') x=(x<<1)+(x<<3)+(ch^48), ch=getchar();
	return x*f;
}

void add(int u,int v,int w){
	e[++cnt].to=v; e[cnt].next=head[u]; e[cnt].w =w; head[u]=cnt;
	e[++cnt].to=u; e[cnt].next=head[v]; e[cnt].w =0; head[v]=cnt;
}
bool bfs(){
	queue<int>q;
	memset(dep,0,sizeof dep);
	q.push(s); dep[s]=1; now[s]=head[s];
	while(!q.empty()){
		int u=q.front() ; q.pop() ;
		for(int i=head[u];i;i=e[i].next ){
			int v=e[i].to ;
			if(dep[v]==0&&e[i].w ){
				dep[v]=dep[u]+1;
				now[v]=head[v];
				if(v==t) return 1;
				q.push(v);
			}
		}
	}
	return 0;
}

int dfs(int u,int lim){
	if(u==t) return lim;
	int flow=0;
	for(int i=now[u];i&&lim>flow;i=e[i].next){
		int v=e[i].to ;
		int w=e[i].w ;
		now[u]=i;
		if(w&&dep[v]==dep[u]+1){
			int res=dfs(v,min(w,lim-flow));
			e[i].w-=res;
			e[i^1].w+=res;
			flow+=res;
		}
	}
	if(flow==0) dep[u]=0;
	return flow;
}

int dinic(){
	int ans=0;
	while(bfs()) ans+=dfs(s,inf);
	return ans;
}
int main(){
	n=read(), m=read(), s=read(), t=read();
	for(int i=1;i<=m;i++){
		int u=read(),v=read(),w=read();
		add(u,v,w);
	}
	printf("%d",dinic());
}
```

##### 无源汇上下界可行流
- **建图**：
对于新图的每个边 $f'(u,v)=c_u(u,v)-c_l(u,v)$
每个点 $A[\ i\ ]= C_{in} -C_{out}$ 
 对于每个 $A[\ i\ ]<0$ 的点u 建一条v-->T
 对于每个 $A[\ i\ ]>0$ 的点v 建一条S-->u
 ```add(u,v,c,d);		A[u]-=c; A[v]+=c; //c是下界```
- 容量限制：$c_l(u,v)<=f(u,c)<=c_u(u,v)$
流量守恒：
在新图中 $c(s,u)= \delta=c_{in}-c_{out}$
对于新图流量守恒 有 $\delta+y=x$ 
即$y+c_{in}-c_{out}=x$
移项得$y+c_{in}=x+c_{out}$
则原图流量守恒得证

- 只有**当新图中与$S$相连的所有边满流**，则对应的，即**使原图中** $C_入 -C_出 =0$ ，才能使原图遵循流量守恒， 因此 —— **$G_f$ 的最大流为 $G$ 的可行流**

##### 最大密度子图
- 图的密度：$$g=\frac{|E'|}{|V'|}$$
$Maximize:h(g)=|E'|-g|V'|$ 一般使用二分求解
转化一下 $Minimize: h(g)=g|V'|-|E'|$

$$
\begin{aligned}
h(g)
&=g|V'|-|E'| \\
&=\sum g- \sum 1\\
&=\sum g-(\frac{\sum dv}{2}+\frac{c[v',\overline v']}{2} ) \\
&=\frac{1}{2}(\sum 2g-\sum dv+c[v',\overline v']) \\
&=\frac{1}{2}(\sum ( 2g-dv)+c[v',\overline v'])
\end{aligned}
$$

- **建图**：
可考虑以 $2g-dv$ 作为边权 由于该值可能为负，因此考虑给所有边加上一个极大值U
$$ \left\{
\begin{matrix}
 c(u,v)=1 \\ 
 c(s,v)=U \\
 c(v,t)=U+2g-d_v \\
\end{matrix} \\
\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \  U=m
\right.
$$

根据割的定义
$$
\begin{aligned}
c[S,T]
&=\sum _{v \in \overline v'} U +\sum _{u \in \overline v'} (2g-dv+U)+\sum _{v\in  v'} \sum _{u \in \overline v'} C_{u,v} \\
&=Un+2g|V'|-2|E'| \\
\end{aligned}
$$
//此处省略一堆证明过程
$$h(g)=g|V'|+|E'|=\frac{Un-c[S,T]}{2}$$

由此得出二分判定的条件： 
如果h(g)>0 则g偏小，答案在右区间， 如果h(g)<0 则g偏大，答案在左区间

- 若此问题**拓展到带边权点权**：

则图的密度定义为：$$g=\frac{\sum W_e}{\sum P_v}$$

通式：$$c[S,T]=U·n+2g|V'|-2\sum_{v\in V'} P_v -2\sum_{e\in E'} W_e$$

###### 带边权最大密度子图
- **建图**：
$$ \left\{
\begin{matrix}
 c(u,v)=W_e \\ 
 c(s,v)=U \\
 c(v,t)=U+2g-d_v \\
\end{matrix} \\
\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ U=\sum _{e\in E}W_e \\
\right.
$$
要注意原图中的边用原图的边权W
此处dv的含义有所改变：表示点v的所有出边的边权和
$$
d_v=\sum _{(u,v)\in E}W_e
$$

###### 带点权最大密度子图
- **建图**：
$$ \left\{
\begin{matrix}
 c(u,v)=1 \\ 
 c(s,v)=U \\
 c(v,t)=U+2g-2p_v-d_v \\
\end{matrix} \\
\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \\ \ \ \ \ \ \ \ \ \\\ \\ \ \ \ \ U=\sum _{v\in v'}P_v \\
\right.
$$
此处pv的含义：即为点v的点权


###### 点边均带权的最大密度子图
- **建图**：
$$ \left\{
\begin{matrix}
 c(u,v)=W_e \\ 
 c(s,v)=U \\
 c(v,t)=U+2g-2p_v-d_v \\
\end{matrix} \\
\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ U=\sum _{v\in v'}P_v+\sum _{(u,v)\in E}W_e \\
\right.
$$
可以将 $2(g-P_v)$理解为平均点权

**总结**：
$$c[S,T]=U·n+2g|V'|-2\sum_{v\in V'} P_v -2\sum_{e\in E'} W_e$$
最小割的表达式通式，其实也就是建图是每个点连向汇点的边权，关键结论记这个就好了
若没有点权则没有$P_v$那一项 ，若没有边权则 $W_e=1$

##### 最小权点覆盖集
**最小权点覆盖集即能覆盖整个图所有边的点权最小的点集。**
注意：**覆盖边指的是该边上至少有一个点被选中，而不是要求该边的两个点都被选中。**
求一个图的最小权点覆盖集是一个NP完全问题，但在**二分图**上有较为优美的做法。
二分图中，==**最小权点覆盖集=最大匹配**==
一般点权都是非负，如果遇到负点权可以类比在“网络战争”中的做法，加上所有负点权值，再在原图中删去点。
- **建图**：
$$ \left\{
\begin{matrix}
 c(u,v)= \infty, v \in S, u \in T  \\ 
 c(s,v)=w , \ \ \  \ \ \ \  \ \ \  \  \  \ v \in S \\
 c(u,t)=w , \ \ \  \ \ \ \  \ \ \  \  \  \  u \in T \\
\end{matrix} \\
\right.
$$
本图中的最小割是简单割 （简单割指与ST点直接相连的割）

- **求方案**：（注意最小权点覆盖集求方案与前面的都不一样）

#### 费用流模型
基于EK算法，将BFS换成SPFA
最小费用跑最短路，最长费用跑最长路，dis[]含义为费用
一般在综合题当中，费用为点或边的权值，而对点边的限制体现在最大流的流量限制
费用流相比最小割较为简单
- 题目要求输出最小费用和最大费用时，可以先跑一遍最小费用，还原容量之后将费用边权全部取反，再跑一遍最小费用并给结果取反，就是最大费用。免去了两遍EK的繁琐。
- 费用流求方案就看题意随机应变好了
从反向边搜，容量大于0的说明正向边有流量，正向边的费用边权×流量就是该边的费用

```cpp
#include<iostream>
#include<cstdio>
#include<queue>
#include<cstring>
#define inf 0x3f3f3f3f
using namespace std;

const int N=5010,M=100100;

int n,m,S,T,cnt=1;
int dis[N],vis[N],incf[N],pre[N],head[N];
struct node{
	int to,next,w,c;
}e[M];

int read(){
	int x=0,f=1; char ch=getchar();
	if(ch=='-') f=-1, ch=getchar();
	while(ch<'0'||ch>'9') ch=getchar();
	while(ch>='0'&&ch<='9') x=(x<<1)+(x<<3)+(ch^48), ch=getchar();
	return x*f;
}

void add(int u,int v,int w,int c){
	e[++cnt].to =v; e[cnt].next =head[u]; e[cnt].w=w; e[cnt].c=c; head[u]=cnt;
	e[++cnt].to =u; e[cnt].next =head[v]; e[cnt].w=0; e[cnt].c=-c; head[v]=cnt;
}

bool spfa(){
	queue<int>q;
	memset(dis,0x3f,sizeof dis);
	memset(incf,0,sizeof incf);
	q.push(S); dis[S]=0; incf[S]=inf;
	while(!q.empty()){
		int u=q.front(); q.pop();
		vis[u]=0;
		for(int i=head[u];i;i=e[i].next ){
			int v=e[i].to ;
			if(e[i].w&&dis[v]>dis[u]+e[i].c){
				dis[v]=dis[u]+e[i].c;
				pre[v]=i;
				incf[v]=min(e[i].w,incf[u]);
				if(!vis[v]){
					q.push(v);
					vis[v]=1;
				}
			}
		}
	}
	return incf[T]>0;
}

void EK(int& flow, int& cost){
	flow=cost=0;
	while(spfa()){
		int t=incf[T];
		flow+=t; cost+=t*dis[T];
		for(int i=T;i!=S;i=e[pre[i]^1].to){
			e[pre[i]].w-=t;
			e[pre[i]^1].w+=t;
		}
	}
}

int main(){
	n=read(); m=read(); S=read(); T=read();
	for(int i=1;i<=m;i++){
		int u=read(), v=read(), w=read(), c=read();
		add(u,v,w,c);
	}
	int flow,cost; 
	EK(flow,cost);
	printf("%d %d",flow,cost);
	
	return 0;
}
```
